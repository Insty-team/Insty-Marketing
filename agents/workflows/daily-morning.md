# 매일 아침 루틴

> CEO 오케스트레이터에게 이 워크플로우를 실행시키면
> 3개 팀이 병렬로 아침 업무를 수행합니다.

## 실행 방법
```
@agents/prompts/ceo-orchestrator.md
아침 루틴 실행해줘.
```

## CEO가 수행할 작업

### 1. 리서치팀 (병렬)
- 커뮤니티 모니터링: AI 관련 새 글/질문 확인
- 잠재 리드 1~3명 식별
- 결과물: 리드 리스트 + 접근 방법

### 2. 영업팀 (병렬)
- CRM 확인: 오늘 팔로업할 리드
- 어제 보낸 DM 응답 확인
- 신규 리드에게 1차 DM 작성

### 3. 마케팅팀 (병렬)
- 오늘의 콘텐츠 카테고리 확인 (월~금 로테이션)
- 콘텐츠 1개 작성 (English + 한국어)
- 커뮤니티 참여 포스트 작성 (해당 요일에)

## 완료 후 CEO 보고
- 📊 오늘의 파이프라인 현황 (리드 수, 단계별)
- 📝 오늘 할 일 요약 (사장이 직접 해야 할 것 표시)
