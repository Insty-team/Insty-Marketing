# 주간 콘텐츠 생산

> 마케팅팀 집중 워크플로우 — 1주일치 콘텐츠를 한 번에 생성

## 실행 방법
```
@agents/prompts/marketing.md
이번 주 콘텐츠 일괄 생산해줘.
이번 주 테마: [테마]
주요 성과/소식: [공유할 내용]
```

## 생산할 콘텐츠

### English (빌드인퍼블릭 — 북미)
| 요일 | 유형 | 채널 |
|------|------|------|
| 월 | 이번 주 목표 | X 싱글 |
| 화 | AI 도구 팁 | X 쓰레드 (5~7) |
| 수 | 성과/사례 | LinkedIn 롱폼 |
| 목 | 1인 사업 인사이트 | X 싱글 |
| 금 | 주간 회고 | X 쓰레드 + LinkedIn |

### 한국어 (세일즈 지원)
| 요일 | 유형 | 채널 |
|------|------|------|
| 화 | AI 활용 팁 | X + 브런치 |
| 목 | 고객 사례/인사이트 | X + 커뮤니티 |

## 출력 형식
각 콘텐츠를 바로 복사+붙여넣기 가능한 형태로 출력.
해시태그, 이모지 포함. 이미지 필요 시 프롬프트 제안.
