# 신규 고객 온보딩

> 영업 + 딜리버리 협업 워크플로우

## 실행 방법
```
@agents/prompts/ceo-orchestrator.md
신규 고객 온보딩 시작.
고객명: [이름]
패키지: [스타터/그로스/프리미엄]
비즈니스: [고객 비즈니스 설명]
킥오프 미팅 노트: [내용]
```

## 단계 1: 영업팀 (즉시)
- 고객 정보 CRM 업데이트 (단계: "계약 완료")
- 킥오프 미팅 아젠다 준비
- 환영 메시지 작성

## 단계 2: 딜리버리팀 (킥오프 후)
- 고객 폴더 생성 (`clients/[고객명]/`)
- 킥오프 노트 → brief.md 정리
- 구축 일정표 작성
- Day 1~7 작업 계획

## 단계 3: CEO 보고
- 프로젝트 타임라인
- 사장이 직접 해야 할 것 (미팅, 결제 확인 등)
- 리스크 요인
