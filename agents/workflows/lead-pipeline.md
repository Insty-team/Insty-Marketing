# 리드 발굴 → 전환 파이프라인

> 리서치 → 영업 순차 실행 워크플로우

## 실행 방법
```
@agents/prompts/ceo-orchestrator.md
리드 파이프라인 실행해줘. 타겟: [세그먼트명]
```

## 단계 1: 리서치팀 (먼저 실행)
- 지정된 세그먼트에서 잠재 고객 10명 조사
- 각 리드별: 이름, 채널, 비즈니스, 고민/니즈, 접근 포인트
- 우선순위 정렬 (전환 가능성 높은 순)

## 단계 2: 영업팀 (리서치 결과 받은 후)
- 상위 5명에 대해 개인화된 1차 DM 작성
- 각 DM에 그 사람의 구체적 상황 반영
- CRM에 리드 등록 (단계: "1차 DM 발송")
- 팔로업 일정 설정 (3일 후)

## 단계 3: CEO 보고
- 리드 10명 요약표
- DM 5개 (사장 검토 후 발송)
- 팔로업 캘린더
