# 고객 납품용 AI 에이전트 팀

> 1인 사업자가 Claude Code (또는 ChatGPT/Gemini)로 직접 운영할 수 있는
> AI 에이전트 팀 시스템. 우리가 구축해주고, 고객이 직접 운영.

## 이게 뭔가요?

혼자 사업하면 마케팅, 영업, 고객 관리, 리서치를 다 혼자 해야 합니다.
AI 에이전트 팀을 만들면 각 역할을 AI에게 맡기고,
**나는 의사결정과 고객 대면에만 집중**할 수 있습니다.

## 어떻게 작동하나요?

```
나 (사장)
  └→ "이번 주 콘텐츠 만들고, 리드 5명 찾아줘"
      └→ CEO 에이전트가 업무 분배
          ├→ 마케팅 에이전트: 콘텐츠 5개 생성
          ├→ 리서치 에이전트: 리드 5명 + 분석
          └→ 영업 에이전트: 맞춤 DM 작성
      └→ CEO가 결과 종합해서 보고
```

## 포함된 에이전트

| 에이전트 | 하는 일 | 사용 도구 |
|---------|--------|----------|
| CEO | 업무 분배, 결과 통합, 보고 | Claude Code |
| 마케팅 | 콘텐츠 생성, SNS, SEO | Claude / ChatGPT |
| 영업 | DM, 제안서, CRM 관리 | Claude / Notion |
| 리서치 | 시장 조사, 경쟁 분석, 트렌드 | Claude / 웹 검색 |

## 사용법

### 기본 (ChatGPT/Claude 무료~유료)
프롬프트 파일을 복사해서 대화 시작 시 붙여넣기.
```
[마케팅 에이전트 프롬프트 복사]
→ ChatGPT에 붙여넣기
→ "오늘 인스타 게시물 만들어줘"
```

### 고급 (Claude Code 사용자)
프롬프트를 `@` 참조로 자동 로드.
```
@agents/prompts/ceo-orchestrator.md
아침 루틴 실행해줘.
```

## 업종별 템플릿

`templates/` 폴더에 업종별 맞춤 프롬프트가 있습니다:
- `solopreneur-general.md` — 1인 사업자 범용
- `coach-consultant.md` — 코치/컨설턴트
- `creator-instructor.md` — 크리에이터/온라인 강사
- `ecommerce.md` — 소규모 이커머스
- `freelancer.md` — 프리랜서 (디자인/개발/번역)
