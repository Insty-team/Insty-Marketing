## 1. 비전

## 2. 전략

세 트랙을 동시에 운영한다.

**한국 대행 (현금)**: 비개발자 사업자의 비즈니스 문제를 AI로 해결하고 돈을 받는다. 직접 소통, 직접 딜리버리. 무료 1건만 하고 M2부터 유료 전환.

**글로벌 대행 (고단가 현금)**: Upwork과 Twitter에서 영어로 AI 자동화 프로젝트 수주. 글로벌 1건($1,500-3,000)은 한국 3-4건과 같다. M3부터 시작.

**글로벌 콘텐츠 (자산)**: 직장생활 공감 썰을 영어 쇼츠로 생산. 유유미미 스타일 — 일러스트 + voice over + text over. 세일즈 아닌 순수 엔터테인먼트. 프로필에 서비스 링크만. DALL-E + ElevenLabs + ffmpeg 자동 파이프라인. 얼굴 노출 없음.

대행 1건을 하면 세 겹의 가치가 쌓인다:
1. **돈** (단기 현금)
2. **콘텐츠 소재** (중기 자산)
3. **솔루션 DB +1** (장기 — Agent 마켓플레이스의 씨앗)

## 3. 인생 로드맵 (35-43세)

| 나이 | 시기 | 월 수입 | 누적 자산 | 사업 가치 | 인생 이벤트 |
|:----:|:----:|:-------:|:---------:|:---------:|:-----------|
| 35 | 지금 | 445만 | 0 | 0 | 사이드 시작 |
| 36 | Y1 | 800-1,000만 | 약 5,000만 | 5-10억 | 글로벌 대행 궤도, 데이팅 |

| 39 | Y4 | 2,500-3,500만 | 약 4억 | 35-50억 | **퇴사**, 주 30-35h |

> **자산 = 현금 + 부동산 + 투자 + 사업 가치.** 현금 7억이 아니라 총 자산 50-70억.

## 4. Phase 정의

### Phase 1: 씨앗 (M1-6) — 첫 수입 + 시스템 구축

| 핵심 행동 | 상세 |
|-----------|------|
| 한국 대행 | 무료 1건 → M2부터 유료 (30-80만/건) |
| 글로벌 대행 | M3부터 Upwork ($50/h → $100/h) |
| 콘텐츠 | 영어 쇼츠 주 3편, 파이프라인 완성 |

**종료 기준**: 월 합산 800-1,000만, 쇼츠 70편+, Upwork 리뷰 3개+

**안 되면?**

| 상황 | 대응 |
|------|------|
| 한국 대행 2건 미만 | 타겟 변경 (1인 → 에이전시) or 크몽/숨고 등록 |
| Upwork 수주 0건 | 시급 $30으로 낮춰서 첫 리뷰 확보, 이후 올리기 |
| 쇼츠 조회수 200 미만 | 소재 피봇 (직장 공감 → AI 팁), 포맷 피봇 (일러스트 → 화면 녹화) |
| 번아웃 | LIGHT 모드 (주 10h), 1주 완전 오프 |

### Phase 2: 가속 (M7-18) — 월 1,000만+ 안정화

| 핵심 행동 | 상세 |
|-----------|------|
| 한국 대행 | 월 3-5건, 단가 50-100만 |
| 글로벌 대행 | 월 1-2건, $2,000-3,000/건 |
| 패턴 발견 | 반복 패턴 3개+ → 가이드/템플릿 출시 |
| 수익화 | 구독자 1,000명 돌파 → 유튜브 광고 수입 |

**종료 기준**: 월 합산 1,500만+, 구독자 5,000+, 패턴 3개+, 커뮤니티 50명+

**안 되면?**

| 상황 | 대응 |
|------|------|
| 대행 매출 정체 (월 300만 이하) | 글로벌 비중 올리기 (달러 단가가 핵심 레버) |
| 패턴이 안 보임 | 대행 10건 넘기면 반드시 보임. 기록 습관 점검 |
| 구독자 정체 | 콘텐츠 스타일 A/B 테스트, TikTok 병행, 협업 |
| 시간 부족 | 한국 대행 축소 → 글로벌 고단가에 집중 (같은 시간, 3배 수입) |

### Phase 3: 전환 (M19-30) — 노동에서 시스템으로

| 핵심 행동 | 상세 |
|-----------|------|
| 유료 커뮤니티 | 글로벌 $49-99/월, 한국 5-10만/월, 50명+ |
| 워크숍 | 월 1-2회, 30-50만/인 |
| 대행 선별 | 고단가 건만 (글로벌 $3,000+, 한국 100만+) |
| 투자 시작 | All Weather 자동 적립 (월 수입의 20-30%) |

**종료 기준**: 패시브 30%+, 월 합산 2,000만+, 전세 계약 완료

**안 되면?**

| 상황 | 대응 |
|------|------|
| 커뮤니티 유료 전환 실패 | 가격 낮추기 ($29), 무료 기간 연장, 가치 재설계 |
| 전세 자금 부족 | 정규직 유지 + 사이드로 2년 더 모으기 |
| 결혼 상대 못 만남 | 사이드 주 12h로 줄이고, 소셜 시간 늘리기 |
| 사업과 연애 둘 다 정체 | 3개월 사업 LIGHT 모드 → 인생 리셋 기간 |

### Phase 4: 자립 (M31-48) — 퇴사 + 여유

| 핵심 행동 | 상세 |
|-----------|------|
| 퇴사 | 사이드가 정규직의 2배 달성 시 |
| 시간 | 주 30-35h (풀타임이지만 여유) |
| 자가 전환 | 전세에서 매매 검토 |
| 투자 본격화 | All Weather 월 500만+ 자동 적립 |
| 커뮤니티 자생 | 모더레이터 운영, 멤버끼리 가치 생성 |

**종료 기준**: 주 30h, 월 합산 2,500만+, 자가 마련, 결혼

**안 되면?**

| 상황 | 대응 |
|------|------|
| 퇴사 후 수입 급감 | 비상금 6개월분 유지, SLAM 정규직 복귀 가능 (안전망) |
| 사업 성장 정체 | 새 수익원 탐색 (Agent 마켓, 교육, 에이전시 스케일) |
| 부동산 가격 폭등 | 지역 변경 (판교/정자), 또는 전세 유지 |

### Phase 5: 자유 (Y5-8) — 경제적 자유 + 기여

| 핵심 행동 | 상세 |
|-----------|------|
| 일하는 시간 | 주 25-30h |
| 수입 | 월 3,000-5,000만 (패시브 60-80%) |
| 투자 | All Weather 자동, 사업 이익 40% 재투자 |
| 비전 실현 | "개인 자생 지원" 인프라 완성 (커뮤니티 + Agent 공간) |
| 기여 | 커뮤니티에서 다른 사람들도 같은 경로를 밟을 수 있도록 |

**안 되면?**

| 상황 | 대응 |
|------|------|
| 100억 미달 (Y8에 30-50억) | 그래도 충분히 윤택. 목표 시점을 Y10-12로 연장 |
| AI 시장 급변 | 사업 형태 피봇, "사람 연결 + 검증된 경험" 가치는 유지 |
| 건강 문제 | 주 20h로 축소, 시스템이 돌아가므로 가능 |

## 11. Agent 공간 실험 (취미)

> 장기 비전의 씨앗. 대행하면서 틈틈이, 취미처럼.

**궁극 비전**: "개인과 그들의 Agent가 자생할 수 있는 공간 — **Insty**"

| 시기 | 실험 | 쌓이는 자산 |
|------|------|-----------|
| M1-3 | 대행 솔루션을 JSON으로 구조화 저장 | 솔루션 DB 5-10건 |
| M4-6 | MCP 서버로 expose (POC) | DB 15-20건 + POC |
| M7-12 | 고객 비즈니스 Agent 제작, 상호작용 테스트 | 통신 경험 + DB 30건+ |
| M13-18 | 솔루션 API + 조회당 과금 테스트 | Agent 마켓 MVP |
| M19-24 | Agent 등록 → 발견 → 거래 | 마켓플레이스 초기 |
| M25-36 | 시장 타이밍 판단 | 스케일 or 보존 |

```
100억 시나리오:
  솔루션 DB 500건 + Agent 1,000개 + 월 거래 10,000건
  → ARR $6M → 밸류 $60M (약 800억)

씨앗은 오늘 심는다: 대행 1건 → JSON → 솔루션 DB +1
```

| M3 | 첫 수입 | 유료 대행 2건+ | 무료 범위 확대, 크몽 등록, Upwork 시급 낮추기 |
