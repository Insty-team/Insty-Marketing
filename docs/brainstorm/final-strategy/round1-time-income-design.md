## 1. 시간 배분 설계

### 대행 1건 = 몇 시간?

대행 작업을 구체적으로 분해하면:

| 단계 | 작업 내용 | 소요 시간 |
|------|-----------|-----------|
| 상담 | 고객 문제 파악, Zoom/카톡 미팅 | 1시간 |
| 분석 | 비즈니스 현황 진단, AI 솔루션 설계 | 2시간 |
| 구축 | Claude Code로 자동화 파이프라인 빌드 | 4-6시간 |
| 테스트 | 고객과 함께 테스트, 수정 | 1-2시간 |
| 인수인계 | 사용법 안내, 문서화 | 1시간 |
| **합계** | | **9-12시간/건** |

> 초기에는 건당 12시간, 템플릿 쌓이면 건당 6-8시간으로 단축 예상.

### 콘텐츠 1개 = 몇 시간?

유유미미 스타일 쇼츠 (일러스트 + voice over + text over):

| 단계 | 작업 내용 | 소요 시간 |
|------|-----------|-----------|
| 소재 발굴 | 직장생활 에피소드 브레인스토밍 | 15분 |
| 스크립트 | 30-60초 분량 스크립트 작성 | 20분 |
| 일러스트 | DALL-E로 장면별 이미지 생성 (3-5장) | 20분 |
| 보이스오버 | ElevenLabs로 영어 나레이션 생성 | 10분 |
| 편집 | ffmpeg으로 이미지+음성+자막 합성 | 15분 |
| 업로드 | 제목/설명/태그 작성, 예약 업로드 | 10분 |
| **합계** | | **1.5시간/편** |

> 파이프라인 자동화 완성 후: **0.5시간/편** (소재 입력 → 반자동 생산)

## 3. 대행 수입 구체화

### 타겟: 한국 비개발자, 누구?

**1차 타겟 (가장 가능성 높은 순)**

| 세그먼트 | 규모 (추정) | 페인포인트 | AI 대행으로 풀 수 있는 것 |
|----------|------------|-----------|------------------------|
| 1인 마케팅 에이전시 | 10만+ | 반복 작업 과다, 시간 부족 | 콘텐츠 자동 생성, 리포트 자동화 |
| 프리랜서 (디자이너/영상) | 20만+ | 영업/세일즈 체계 없음 | 포트폴리오 자동화, 리드 수집 |
| 온라인 강사 (클래스101/탈잉) | 5만+ | 수강생 관리, 마케팅 | 수강생 자동 응대, SNS 자동 포스팅 |
| 스마트스토어/쇼핑몰 | 50만+ | 상품 설명, 고객 응대 | 상품 설명 자동 생성, CS 챗봇 |
| 코치/컨설턴트 | 3만+ | 고객 유치, CRM | 리드 자동화, 예약/결제 시스템 |

### 구체적 문제 5개 (= 서비스 메뉴)

**1. "SNS 콘텐츠 자동 생성"** — 가장 수요 높음
- 고객: 매일 인스타/블로그에 뭘 올릴지 모르는 1인 사업자
- 딜리버리: 업종/키워드 입력 → 주 5개 포스트 자동 생성 파이프라인
- Claude Code + Notion + 자동 예약
- 소요: 6시간 (첫 세팅), 월 1시간 (유지보수)

**2. "고객 문의 자동 응대"** — 즉각적 체감 가치
- 고객: 카톡/DM 문의가 많은데 일일이 답하기 힘든 사업자
- 딜리버리: FAQ 기반 AI 챗봇 세팅 (카카오 채널 or 인스타 DM)
- 소요: 8시간

**3. "리드 수집 + CRM 자동화"** — B2B형 사업자
- 고객: 잠재 고객 관리가 안 되는 컨설턴트/코치
- 딜리버리: Notion DB + 자동 리드 추적 + 팔로업 알림
- 소요: 6시간

**4. "보고서/리포트 자동 생성"** — 고단가
- 고객: 매주/매월 리포트 만드는 데 시간 쓰는 마케터
- 딜리버리: 데이터 소스 연결 → AI 분석 → 자동 리포트 생성
- 소요: 10시간

**5. "비즈니스 프로세스 자동화"** — 맞춤형
- 고객: 반복적 수작업이 많은 모든 1인 사업자
- 딜리버리: 업무 프로세스 분석 → 자동화 가능 부분 AI로 대체
- 소요: 8-12시간

### 가격 설계

| 단계 | 시점 | 가격 | 전략 |
|------|------|------|------|
| 무료 진단 | Month 1-2 | ₩0 | 포트폴리오 + 후기 수집 |
| 소액 테스트 | Month 2-3 | ₩10-20만 | "AI 진단 + 미니 자동화 1개" |

|------|-------------------|-----------|------------|
| Month 1-3 (초기) | 8시간 | 12시간 | 2-3건 |
| Month 4-6 (템플릿화) | 8시간 | 8시간 | 3-4건 |

### 대행에서 추출하는 자산

매 건마다 아래를 반드시 기록:

1. **콘텐츠 소재**: "이런 문제를 이렇게 풀었다" → 쇼츠/포스트 소재
2. **패턴**: 같은 유형의 문제가 반복되면 → 솔루션 템플릿화
3. **템플릿**: 재사용 가능한 코드/설정 → 다음 건 딜리버리 시간 단축
4. **후기/사례**: 고객 허락 받고 → 포트폴리오/프로필에 게시
5. **니즈 데이터**: 고객이 원하는 것 vs 실제 필요한 것 → 솔루션 방향

### 쇼츠 CPM 현실

| 시나리오 | CPM (1,000회당) | 월 100만 조회 기준 월수입 |
|----------|----------------|----------------------|
| 보수적 | $0.02 | $2,000 = ₩260만 |
| 기본 | $0.04 | $4,000 = ₩520만 |
| 낙관 | $0.07 | $7,000 = ₩910만 |

> 쇼츠 CPM은 매우 낮다. 월 100만 조회에 260만-910만원.
> 월 100만 조회 = 일 평균 3.3만 조회 = 쇼츠 1개당 평균 6,600회 (일 5편 기준)

### "콘텐츠 직접 수입" vs "콘텐츠가 대행 고객을 데려오는 것"

| 비교 항목 | 직접 수입 (광고/스폰서) | 간접 수입 (대행 유입) |
|----------|----------------------|-------------------|

| 확장성 | 구독자 비례 (복리) | 시간 비례 (선형) |
| 통제 가능성 | 알고리즘 의존 | 내가 영업 가능 |
| **결론** | 장기 자산 | 단기 캐시플로우 |

> **전략**: Month 1-12는 콘텐츠의 간접 효과(대행 유입)에 집중.
> 콘텐츠 직접 수입은 Month 12 이후 보너스로 취급.
> 콘텐츠의 진짜 가치 = "나를 증명하는 포트폴리오" + "자산으로 쌓이는 조회수"

## 핵심 정리

### Sam이 기억할 숫자

| 항목 | 숫자 |
|------|------|
| 주간 시간 | 대행 8h + 콘텐츠 5h + 자동화 3h + 리서치 2h = 18h |
| 대행 1건 소요 | 초기 12h → 템플릿화 후 6-8h |
| 쇼츠 1편 소요 | 1.5h → 파이프라인 후 0.5h |
| 무료→유료 전환 | Month 3 (무료 3건 후) |
| 대행 Month 12 목표 | 월 200-350만원 (3-4건) |
| 콘텐츠 Month 12 목표 | 구독자 1,000-5,000명, 수입 ₩5-30만 |
| 3년 후 총 자산 | 보수적 1억 / 기본 2억 / 낙관 2.1억 |
| 3년 후 필요 금액 | ~1억 (전세+결혼+비상금) |

### 핵심 인사이트

1. **대행이 캐시카우**: 콘텐츠 수입이 의미 있으려면 최소 12개월. 그 전까지 대행이 동기부여 + 현금흐름 담당.
2. **콘텐츠는 자산**: 쇼츠 1편 = 영원히 조회수를 만드는 자산. 300편 쌓이면 복리 효과.
3. **자동화가 핵심**: 쇼츠 1.5h→0.5h 단축이 주 3편 기준 3시간 절약. 그 시간을 대행에 투입.
4. **무료 진단이 첫 단추**: 돈 안 받고 3건 해주면 → 후기 + 포트폴리오 + 패턴 학습.
5. **듀얼 트랙의 시너지**: 대행 사례 → 콘텐츠 소재, 콘텐츠 팔로워 → 대행 고객 유입.
