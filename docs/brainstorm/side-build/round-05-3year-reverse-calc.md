> 핵심 질문: "3년 후(2029년 3월) 거주 공간 확보 + 결혼이 가능한 재정 상태를 만들려면?"

## 2. 현재 상태에서 3년 저축 시나리오

### Scenario A: 정규직만 (사이드 없음)

| 항목 | 월 금액 | 3년 합계 |
|------|--------|---------|
| 월 저축 | 160만원 | 5,760만원 |
| 상여금/보너스 (연 100만 가정) | ~8만/월 | 300만원 |
| **합계** | | **~6,060만원** |

> **결론**: 1억 필요한데 6,060만원만 모임. **약 4,000만원 부족.**
> 양가 지원이 없으면 정규직만으로는 부족하다.

### Scenario B: 정규직 + 사이드 (보수적)

| 항목 | 월 금액 | 3년 합계 |
|------|--------|---------|
| 정규직 저축 | 160만원 | 5,760만원 |
| 사이드 수입 (평균) | 200만원 | 7,200만원 |
| 사이드 세금 (약 20%) | -40만원 | -1,440만원 |
| 상여금 | ~8만/월 | 300만원 |
| **합계** | | **~11,820만원** |

> **사이드 월평균 200만원이면 목표 달성 가능!**
> 하지만 "평균 200만원"이 되려면 3년간 점진적 성장 필요

### 사이드 수입 성장 시나리오 (현실적)

| 기간 | 월 사이드 수입 | 누적 (세후) |
|------|-------------|-----------|
| 1-6개월 | 30만원 | 144만원 |
| 7-12개월 | 100만원 | 480만원 |
| 13-18개월 | 200만원 | 960만원 |
| 19-24개월 | 350만원 | 1,680만원 |
| 25-30개월 | 500만원 | 2,400만원 |
| 31-36개월 | 700만원 | 3,360만원 |
| **3년 사이드 합계** | | **9,024만원** |
| 세금 20% 제외 | | **7,219만원** |

**3년 총 자산 = 정규직 저축 6,060만원 + 사이드 7,219만원 = 약 1억 3,280만원**

> 1억 목표 대비 **3,280만원 여유**. 달성 가능!

## 3. 역산: 사이드 수입 최소 요구치

### "3년 후 1억"을 위한 최소 사이드 수입

정규직 저축으로 6,060만원 모이므로, 사이드에서 **세후 3,940만원** 이상 필요.
세전으로는 약 **4,925만원** (세율 20% 가정).
36개월 평균 = **월 137만원**.

하지만 초기에는 0원이므로, 성장 곡선을 고려하면:

| 시나리오 | 12개월차 월수입 | 24개월차 | 36개월차 | 3년 세후 합계 | 달성? |
|---------|-------------|---------|---------|------------|------|
| 보수적 | 50만 | 150만 | 300만 | 3,600만 | ❌ 부족 |
| 현실적 | 100만 | 300만 | 600만 | 5,760만 | ✅ 달성 |
| 낙관적 | 200만 | 500만 | 1,000만 | 10,200만 | ✅ 여유 |

> **최소 요구**: 12개월차에 월 100만원, 24개월차에 월 300만원 이상

## 4. 어떤 사이드 모델이 이 타임라인에 맞는가?

Round 2-4에서 분석한 모델을 "3년 역산"에 대입:

| 모델 | 12개월 월수입 | 24개월 월수입 | 3년 목표 적합 |
|------|------------|------------|-------------|
| 디지털 상품 포트폴리오 | 100-300만 | 200-500만 | ✅ |
| 마이크로 SaaS | 50-200만 | 200-800만 | ✅ (느리지만 성장 큼) |
| 프리랜싱/대행 | 200-500만 | 300-600만 | ✅ (빠르지만 번아웃) |
| 뉴스레터 | 0-30만 | 50-150만 | ❌ (너무 느림) |
| 커뮤니티 | 0-50만 | 100-300만 | ⚠️ (불확실) |

### 최적 조합: 디지털 상품 + SaaS (Round 2 결론 재확인)

- **0-6개월**: 디지털 상품으로 월 30-50만원 (동기부여 확보)
- **6-12개월**: 디지털 상품 100만 + SaaS 시작 = 월 100만원
- **12-24개월**: 디지털 상품 200만 + SaaS 100만 = 월 300만원
- **24-36개월**: 디지털 상품 200만 + SaaS 400만 = 월 600만원

## 5. 세금 및 법적 고려사항

### 사이드 수입 세금 (한국)

| 연 사이드 수입 | 세금 구조 | 예상 실효세율 |
|-------------|---------|------------|
| 1,000만원 이하 | 종합소득세 기본 | ~6-10% |
| 1,000-2,400만원 | 종합소득세 | ~15-20% |
| 2,400-4,600만원 | 종합소득세 | ~20-25% |
| 4,600만원 이상 | 종합소득세 | ~25-30% |

### 주의사항
1. **사업자 등록**: 연 수입이 일정 이상이면 필수. 간이과세자로 시작 가능
2. **정규직과의 충돌**: 근로계약서에 겸업 금지 조항 확인 필수
3. **해외 수입**: Gumroad/Stripe 수입은 해외 소득. 신고 필수
4. **건강보험**: 사이드 수입이 연 2,000만원 넘으면 추가 건보료 발생 가능

## 8. Round 5 결론

### 핵심 숫자

| 항목 | 값 |
|------|-----|
| 3년 후 필요 금액 | **~1억원** |
| 정규직만으로 모이는 금액 | ~6,060만원 (**4,000만 부족**) |
| 사이드 최소 요구 (세후 3년 합계) | **~4,000만원** |
| 사이드 월평균 (최소) | **~137만원** (성장 곡선 고려 시 12개월차 100만) |
| 사이드 월평균 (현실적 성장) | 12개월 100만 → 24개월 300만 → 36개월 600만 |

### 가능하다. 단, 조건은:

1. **이번 달부터 시작**: 디지털 상품 첫 출시를 4월 안에
2. **12개월 내 월 100만원**: 디지털 상품 + SaaS 초기 수입
3. **24개월 내 월 300만원**: SaaS MRR이 본격 성장
4. **양가 지원은 보너스**: 없어도 되도록 설계
5. **세금 관리**: 초기부터 사업자 등록 + 경비 처리

### "3년 후 결혼"이라는 중간 목표가 Sam에게 주는 것

- **데드라인**: 막연한 "언젠가 100억"이 아니라 "3년 후 1억"이라는 구체적 목표
- **동기부여**: 매월 저축+사이드 수입이 목표에 다가가는 것이 보임
- **현실 검증**: 12개월 내 월 100만원 못 만들면 전략 수정 시점
