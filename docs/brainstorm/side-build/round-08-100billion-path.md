# Round 8: 100억까지의 레벨업 경로

> 작성일: 2026-03-19
> 이전 라운드 핵심: Round 5 — 3년 내 1억 / Round 6 — HookGen 첫 제품 / Round 2 — 하이브리드 전략
> 핵심 질문: "사이드 빌딩으로 시작해서, 언제 어떻게 100억 자산가가 되는가?"

---

## 1. 100억의 의미 해체

100억 원 = 약 $7.5M USD (2026 기준)

### 100억 달성 방법은 3가지뿐

| 방법 | 설명 | 확률 | Sam 가능성 |
|------|------|------|-----------|
| A. 기업 매각 (Exit) | SaaS 회사를 키워서 팜 | 낮음-중 | ★★★★ |
| B. 기업 보유 (Pieter 모델) | 매각 않고 연 수익으로 자산 축적 | 중 | ★★★ |
| C. 지분 가치 | 투자 받아 기업 가치 100억 이상 | 낮음 | ★★ |

### 저축만으로는 불가능 (다시 확인)
- 월 160만 저축 × 12개월 × 30년 = **5.76억** (이자 포함 8-10억 가능)
- 월 500만 저축 × 12개월 × 30년 = **18억** (이자 포함 25-30억)
- **저축만으로는 100억 불가능. 사업 자산(equity)이 핵심.**

## 2. SaaS Valuation 현실 (2026 데이터)

### ARR별 기업 가치 멀티플

| ARR | 멀티플 | 예상 기업 가치 | 비고 |
|-----|--------|-------------|------|
| $100K (1.3억) | 2-3x | $200-300K | 개인 매수자 |
| $500K (6.5억) | 3-4x | $1.5-2M | 소형 펀드 |
| $1M (13억) | 3-5x | $3-5M (40-65억) | 중형 매수자 |
| $2M (26억) | 4-6x | $8-12M (100-160억) | **100억 달성** |
| $5M (65억) | 5-8x | $25-40M | 대형 PE |
| $10M (130억) | 6-12x | $60-120M | 전략적 매수 |

> **핵심**: ARR $2M (월 MRR ~$167K, 약 2,200만원)이면 기업 가치 100억 달성 가능

### 솔로 파운더 할인

- 솔로 운영 SaaS는 멀티플 할인: ARR 대신 **이익의 2-4x**로 평가
- ARR $2M, 마진 80% = 이익 $1.6M → 기업 가치 $3.2-6.4M (42-83억)
- **100억 확실한 달성: ARR $3M+ 또는 이익 $2.5M+**

## 3. 레벨업 경로: 5단계

### Stage 1: Side Builder (현재 ~ 1년)
**목표**: 첫 제품 PMF + 월 MRR $1,000

```
HookGen 론칭 → 디지털 상품 판매 → SaaS 전환
월 수입: 0 → $500 → $1,000
기업 가치: 0
상태: 정규직 + 사이드
```

**전환 조건**: MRR $1,000 안정적 달성 + 두 번째 제품 아이디어 검증

### Stage 2: Indie Hacker (1-3년)
**목표**: 멀티 프로덕트 MRR $5,000-10,000

```
HookGen (MRR $3K) + NicheNewsAI (MRR $2K) + 디지털 상품 ($2K)
월 수입: $5,000-10,000 (650-1,300만원)
기업 가치: $200-500K (2.6-6.5억)
상태: 정규직 + 사이드 (또는 퇴사 검토)
```

**전환 조건**: 사이드 수입 > 정규직 수입 (월 $3,500+)

### Stage 3: Full-time Founder (3-5년)
**목표**: ARR $500K-$1M

```
정규직 퇴사 → 풀타임 빌딩
메인 SaaS (ARR $300-500K) + 서브 프로덕트들 ($200-500K)
월 수입: $40,000-80,000 (5,200만-1억)
기업 가치: $1.5-5M (19-65억)
상태: 풀타임 솔로 또는 소규모 팀 (1-3명)
```

**전환 조건**: ARR $500K+ 안정적 / 정규직 수입의 3x 이상

### Stage 4: Scaling (5-8년)
**목표**: ARR $2-3M

```
B2B 확장 + 엔터프라이즈 고객 + 팀 빌딩 (3-5명)
메인 SaaS ARR $1.5-2M + 서브 프로덕트 $500K-1M
월 수입: $150,000-250,000
기업 가치: $8-20M (100-260억) ← **100억 달성 구간**
상태: CEO + 소규모 팀
```

**전환 조건**: 100억 가치 달성

### Stage 5: Exit or Hold (8-12년)
**두 가지 경로**:

| 경로 | 장점 | 단점 |
|------|------|------|
| **A. Exit (매각)** | 일시에 큰 돈, 새 시작 가능 | 만든 것을 잃음, 세금 |
| **B. Hold (Pieter 모델)** | 지속 수입, 자유, 복리 | 관리 계속 필요, 시장 변화 리스크 |

## 4. Pieter Levels 모델 vs Exit 모델 비교

### A. Hold 모델 (Pieter Levels)

**현황**: 연 $3M+ 수입, 매각 안 함, 추정 자산 $40-60M

| 장점 | 단점 |
|------|------|
| 매년 $3M+ 현금 흐름 | 시장 변화 시 가치 하락 |
| 풀 오너십 유지 | 관리/유지보수 지속 |
| 새 제품 실험 자유 | 일해야 돈이 들어옴 |
| 세금 최적화 가능 | 대규모 자본화 어려움 |

**Sam에게의 의미**:
- ARR $3M 연 수입 → 10년 누적 = $30M (약 400억) 가능
- 하지만 10년 유지가 보장되지 않음

### B. Exit 모델 (Danny Postma)

**사례**: Headlime 8개월 만에 $1M+ 엑싯

| 장점 | 단점 |
|------|------|
| 일시에 큰 자본 확보 | 보통 세금 20-30% |
| 깔끔한 마무리 | 다음 것을 또 만들어야 |
| 새 프로젝트 자금 | 잘못 팔면 후회 |
| 비경쟁 조항 주의 | 조건 협상 어려움 |

**Sam에게의 의미**:
- ARR $2M 기업을 5x에 매각 = $10M (약 130억) → 세후 약 100억
- 하지만 ARR $2M까지 키우는 데 5-8년

### 추천: Hybrid (키우면서 결정)

```
Stage 1-3: 무조건 Hold (키우기)
Stage 4: ARR $2M+ 되면 판단
  ├─ 성장세 강하면 → Hold (Pieter 모델)
  └─ 정체되면 → Exit (매각)
```

## 6. 각 단계의 전환 조건 (체크리스트)

### Side Builder → Indie Hacker
- [ ] MRR $1,000 3개월 연속 달성
- [ ] 두 번째 제품 아이디어 검증 완료
- [ ] 월 10시간 미만으로 유지보수 가능한 제품 1개+

### Indie Hacker → Full-time Founder
- [ ] MRR > 정규직 월급의 1.5x ($5,250+)
- [ ] 6개월 생활비 비상금 확보
- [ ] 성장 추세 3개월 연속 양수
- [ ] 배우자/파트너 동의

### Full-time Founder → Scaling
- [ ] ARR $500K+
- [ ] 채용 1-2명 (고객 지원, 마케팅)
- [ ] B2B 고객 비중 30%+

### Scaling → Exit/Hold 결정
- [ ] ARR $2M+ (기업 가치 100억 가능 구간)
- [ ] 성장률 연 40%+ → Hold
- [ ] 성장률 연 20% 미만 → Exit 검토
- [ ] 번아웃/삶의 변화 → Exit

## 7. 100억을 만드는 "진짜 레버"

### 저축 ≠ 100억. 자산(Equity) = 100억.

| 레버 | 효과 | Sam의 우선순위 |
|------|------|-------------|
| **반복 수익 (MRR)** | 기업 가치의 근간 | ★★★★★ |
| **순 수익 유지율 (NRR)** | 멀티플 30-50% 증가 | ★★★★ |
| **B2B 비중** | 단가 ↑, churn ↓ | ★★★★ |
| **멀티 프로덕트** | 리스크 분산 + 크로스셀 | ★★★ |
| **자동화** | 운영비 ↓, 마진 ↑ | ★★★★★ |
| **브랜드/청중** | 신제품 론칭 시 즉시 트래픽 | ★★★ |

### 가장 중요한 것: NRR (Net Revenue Retention)

NRR > 110% = 기존 고객만으로도 성장
→ 멀티플 1-2x 추가 (= 기업 가치 $5-10M 차이)

**HookGen에서 NRR 높이는 법**:
- $9 → $29 → $79 업그레이드 경로
- 팀 플랜 추가 (에이전시용)
- 사용량 기반 과금 (API 호출)
