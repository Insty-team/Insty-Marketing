## 전략 1: SLAM 글로벌 프리랜싱 — 파괴 테스트

### 약점 1: "첫 프로젝트를 못 잡으면?"

**시나리오**: Upwork에서 SLAM 프로젝트가 올라오지만, 프로필에 리뷰가 0이라 선택되지 않음. 2~3개월간 수주 실패.

**확률**: 중간 (25~35%)

**대응 계획**:
- **전술 A**: 첫 3건은 시장가의 50~70%로 비딩 → 리뷰 확보 후 정상가
- **전술 B**: Upwork 외에 Arc.dev, Freelancer.com 동시 진행
- **전술 C**: LinkedIn에서 로보틱스 스타트업 CTO에게 직접 DM → 플랫폼 우회
- **전술 D**: 오픈소스 SLAM 프로젝트에 기여 → GitHub 포트폴리오 강화
- **피봇**: 3개월 내 수주 실패 시 → 한국 계약직/파트타임으로 전환 (전략 ④)

### 약점 2: "SLAM 프리랜싱 수요가 기대보다 적으면?"

**시나리오**: SLAM 프리랜서 시장이 생각보다 작고, 풀타임급 물량 확보 불가. 월 10~15시간만 가능.

**확률**: 낮음~중간 (15~25%)

**대응 계획**:
- SLAM에 인접한 분야로 확장: ROS2 개발, 센서 퓨전, 컴퓨터 비전
- 이들을 합치면 프로젝트 풀이 5~10배 증가
- **피봇**: SLAM 단독이 아닌 "로보틱스 AI 엔지니어"로 포지셔닝 확장

### 약점 3: "번아웃 — 주 40시간 코딩하면서 콘텐츠도?"

**시나리오**: 프리랜싱 + 콘텐츠 제작 + 네트워킹으로 주 55~60시간 → 6개월 후 번아웃.

**확률**: 높음 (40~50%)

**대응 계획**:
- **핵심**: 프리랜싱 시간을 주 30시간으로 제한 (절대 초과 금지)
- **콘텐츠 효율화**: AI 도구(Claude, Whisper)로 편집 시간 최소화
- **주기적 휴식**: 4주 일 → 1주 슬로우 (Pomodoro 대신 Sprint/Rest 사이클)
- **피봇**: 번아웃 조짐 시 → 콘텐츠 중단, 프리랜싱만 집중 (현금 우선)

### 약점 4: "환율 변동 리스크"

**시나리오**: 원/달러 환율 급변 (예: $1=₩1,100으로 하락)

**확률**: 중간 (변동성은 항상 있음)

**대응 계획**:
- 달러 수입의 일부를 달러로 유지
- 환율 하락 시 생활비가 상대적으로 줄어드는 효과도 있음
- **영향**: 제한적. ₩1,350 → ₩1,100이어도 월 $10K는 ₩1,100만 → 여전히 목표 내

### 약점 5: "AI가 SLAM 엔지니어를 대체하면?"

**시나리오**: Foundation Model 기반 Visual SLAM이 등장해서 전통적 SLAM 엔지니어 수요 감소.

**확률**: 낮음 (3~5년 내에는) → 중간 (5~10년)

**대응 계획**:
- 오히려 기회: "AI + SLAM" 교차 스킬이 더 가치 높아짐
- AI 기반 SLAM 도구를 "사용하고 가르치는" 쪽으로 전환
- **피봇**: SLAM 순수 코딩 → SLAM 아키텍처 컨설팅 + AI 통합으로 이동

## 전략 2: SLAM 특화 교육 비즈니스 — 파괴 테스트

### 약점 1: "오디언스가 안 모이면?"

**시나리오**: 유튜브 6개월 운영했지만 구독자 500명, 영상 평균 200뷰. 교육 상품 판매 부진.

**확률**: 중간~높음 (30~45%)

**대응 계획**:
- **콘텐츠 전략 변경**: 롱폼 → 숏폼 (SLAM 관련 Shorts/Reels)
- **플랫폼 변경**: 유튜브 → 트위터/X (기술 커뮤니티에서 더 빠른 성장 가능)
- **게스트 출연**: 로보틱스 팟캐스트, 기술 유튜버와 콜라보
- **SEO 최적화**: "SLAM tutorial", "ROS2 SLAM" 등 검색 트래픽 노린 콘텐츠
- **피봇**: 6개월 후 견인력 없으면 → 교육은 부업으로 축소, 프리랜싱 집중

### 약점 2: "대학/MOOC가 같은 코스를 무료로 제공하면?"

**시나리오**: Coursera나 MIT OCW에서 SLAM 코스 런칭. 무료/저가 경쟁.

**확률**: 중간 (이미 일부 존재)

**대응 계획**:
- **차별화**: 이론 vs 실무의 차이를 극대화
  - 대학 코스: "SLAM 수학 이론"
  - Sam의 코스: "내일 회사에서 바로 돌리는 SLAM 파이프라인"
- **실습 중심**: 실제 데이터셋, 실제 코드, 실제 문제 해결
- **커뮤니티**: 코스 수강생 전용 Slack/Discord → 지속적 가치
- **결론**: 대학 코스와 실무 코스는 타겟이 다름. 직접 경쟁 아님.

### 약점 3: "SLAM 교육 시장이 너무 작으면?"

**시나리오**: 전 세계 SLAM 학습자가 수만 명밖에 안 되어 교육 비즈니스 천장이 낮음.

**확률**: 중간 (20~30%)

**대응 계획**:
- **확장 1**: SLAM → "로보틱스 AI" 전체로 범위 확대
- **확장 2**: SLAM 코스의 성공을 레버리지로 인접 분야 코스 추가 (센서 퓨전, ROS2, 컴퓨터 비전)
- **확장 3**: 교육이 아닌 도구(SaaS)로 전환 (Round 8의 AI SLAM Tuning)
- **고가 전략**: 시장이 작으면 단가를 올린다. 100명 × $29가 아니라 10명 × $2,999
- **피봇**: 코호트/멘토링 중심 고가 모델로 전환

### 약점 4: "콘텐츠 제작 능력이 부족하면?"

**시나리오**: 영상 촬영/편집에 서투르고, 영어 프레젠테이션이 어색.

**확률**: 중간 (30~40%)

**대응 계획**:
- **AI 도구 활용**: 스크립트(Claude), 편집(Descript), 썸네일(Canva), 음성(자연스러운 영어 TTS 보조)
- **스크린 캐스트 중심**: 얼굴 안 나와도 됨 → 코드/터미널 시연 중심
- **글 먼저**: 영상 어려우면 블로그 + 트위터부터 시작
- **외주**: 편집만 외주 ($100~$300/영상)

## 전략 3: Build in Public + SLAM Interview Prep — 파괴 테스트

### 약점 1: "아무도 관심 없으면?"

**시나리오**: Build in Public 포스트에 반응 없음. 팔로워 100명에 머묾.

**확률**: 중간 (30~40%)

**대응 계획**:
- **커뮤니티 참여 먼저**: 자기 콘텐츠만 올리지 말고, 다른 사람 콘텐츠에 가치 있는 댓글
- **니치 해시태그/커뮤니티**: #robotics, #SLAM, #buildInPublic, Robotics subreddit
- **콜라보**: 다른 Build in Public 하는 사람들과 상호 프로모션
- **핵심**: 6개월은 투자 기간. 반응 없어도 계속 올려라.

### 약점 2: "수입 공개가 부메랑 되면?"

**시나리오**: 수입 공개 후 클라이언트가 "너 이만큼 버니까 단가 낮춰라" 요구.

**확률**: 낮음 (10~15%)

**대응 계획**:
- 총 수입은 공개하되 개별 클라이언트 단가는 비공개
- "가치 기반 가격" 논리 유지
- 오히려 투명성이 신뢰 → 단가 상승 효과가 더 큼

### 약점 3: "Interview Prep 구매자가 없으면?"

**시나리오**: SLAM 면접 준비 서비스를 만들었지만, 수요가 기대에 미치지 못함.

**확률**: 중간 (25~35%)

**대응 계획**:
- **무료 시작**: 유튜브/블로그에 "SLAM 면접 질문 Top 10" 콘텐츠 → 수요 테스트
- **가격 실험**: $29 PDF → $99 코스 → $299 패키지 순서로 상향
- **피봇**: 면접 준비 → SLAM 실무 코스로 전환 (더 넓은 시장)

### 최악의 시나리오: "모든 것이 실패하면?"

```
프리랜싱 실패 + 콘텐츠 실패 + 교육 실패
확률: 5% 미만 (모든 것이 동시에 실패할 확률)

이 경우의 안전망:
1. 한국 SLAM 정규직 (시니어 연봉 ₩1.1억+)
   → SLAM 5년 경험은 취업 시장에서 여전히 프리미엄
   → 3년 × ₩1.1억 = ₩3.3억 (생활비 차감 후 ~₩2.6억)
   → 3.1억에는 약간 부족하지만 근접

2. 한국 자율주행 회사 (42dot, NaverLabs 등)
   → 스톡옵션 포함 시 추가 업사이드

결론: 최악의 경우에도 "SLAM 5년"이라는 안전망이 존재.
     매출 0이지만 실력 0이 아니라는 것이 Sam의 최대 강점.
```

## 주요 리스크 완화 전략

### 1. "안전판" 유지
```
절대 규칙: 월 생활비 6개월분(₩1,200만)은 항상 비상금으로 유지
  → 프리랜싱이 3개월 끊겨도 생존 가능
  → 여유가 있어야 좋은 판단 가능
```

### 2. "빠른 검증"
```
모든 아이디어에 대해:
  Week 1: 최소 버전 만들기 (MVP)
  Week 2-4: 첫 고객/반응 확인
  Month 2-3: 숫자로 판단 (수입, 조회수, 전환율)
  → 3개월 내 견인력 없으면 피봇 or 중단
```

### 3. "순서 관리"
```
절대 하지 말 것: 모든 것을 동시에 시작
반드시 할 것:
  Phase 1 (Month 1-3): 프리랜싱만 집중 → 현금 확보
  Phase 2 (Month 3-6): + Build in Public (시간 추가 주 3시간)
  Phase 3 (Month 6-9): + 교육 콘텐츠 (프리랜싱 시간 줄이기 시작)
  Phase 4 (Month 9-12): + 유료 상품 런칭
```

## Round 9 결론

### 전략의 강점
- **SLAM 5년이라는 안전망**: 모든 것이 실패해도 정규직으로 돌아갈 수 있음
- **리스크가 분산됨**: 단기(프리랜싱) + 장기(교육)로 한쪽이 실패해도 다른 쪽이 버팀
- **피봇이 자연스러움**: 같은 도메인(SLAM/로보틱스) 내에서의 피봇이라 학습 비용 없음

### 가장 큰 리스크
1. **번아웃** (40~50%) — 가장 높은 확률, 반드시 시간 관리
2. **오디언스 성장 실패** (30~45%) — 6개월 인내 필요, 형식/채널 실험
3. **프리랜싱 초기 수주 실패** (25~35%) — 단가 양보 + 다채널 접근으로 극복

### 핵심 방어 원칙
> **"돈이 떨어지기 전에 피봇하라."**
> 비상금 6개월분 유지 + 3개월 주기 검증 + 과감한 피봇 = 실패해도 다시 시작 가능.
