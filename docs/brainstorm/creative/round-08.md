## 아이디어 1: "RoboL10n" — 로봇 UX 현지화 컨설팅

### 컨셉
로봇이 인간과 상호작용할 때의 UX(음성, 제스처, 내비게이션 패턴)를 문화권별로 현지화하는 전문 서비스.

### 왜 존재하지 않나
- 소프트웨어 현지화(번역)는 있지만, **로봇 행동의 현지화**는 새로운 개념
- 로봇이 사람과 상호작용하는 방식은 문화마다 다름:
  - 접근 거리 (한국 vs 미국 vs 일본)
  - 속도와 움직임 패턴
  - 음성 톤과 표현
  - 사용자 인터페이스 디자인

### 왜 Sam만 할 수 있나
- SLAM (로봇 내비게이션 이해) + Localization (문화적 적응 전문성)
- 이 교차점에 있는 사람은 **세상에 거의 없음**

### 수익 모델
- 컨설팅: $10,000~$50,000/프로젝트 (로봇 회사의 해외 시장 진출 시)
- 가이드/프레임워크 판매: $499~$2,999
- 워크숍: $3,000~$10,000/회

### 현실 점검
- **시장 존재 여부**: 아직 없음 → 시장 자체를 만들어야 함
- **타이밍**: 가정용/서비스 로봇이 확산되면 필수 → 2~5년 후 본격화
- **리스크**: 너무 이른 시장
- **점수**: 독특함 ◎◎◎ / 즉시 수익 × / 미래 가치 ◎◎

## 아이디어 2: "SLAM Interview Prep" — SLAM 엔지니어 면접 코칭

### 컨셉
SLAM/로보틱스 엔지니어 지망생의 기술 면접 준비를 돕는 유료 서비스.

### 왜 기회인가
- SLAM 엔지니어 수요 폭발 → 지원자도 증가
- LeetCode 같은 일반 면접 준비는 있지만 **SLAM 특화 면접 준비 서비스는 없음**
- 실무 5년 경험자의 "이런 질문 나오면 이렇게 답해라" = 고가치

### 수익 모델
- 1:1 코칭: $150~$300/session (1시간)
- 패키지 (5회): $500~$1,000
- 자가학습 코스: $99~$199
- 면접 질문/답변 가이드 PDF: $29~$49

### 시뮬레이션
```
월 10명 1:1 코칭 × $200 = $2,000
월 코스 판매 20건 × $149 = $2,980
월 PDF 판매 50건 × $39 = $1,950
────────────────────────────
월 합계: $6,930 (약 ₩930만)
```

### 현실 점검
- **시장 크기**: 글로벌 SLAM 엔지니어 지망생 수만 명
- **경쟁**: 없음 (니치 독점 가능)
- **스케일**: 코스/PDF는 패시브 인컴
- **점수**: 독특함 ◎◎ / 즉시 수익 ○ / 스케일 ◎

## 아이디어 3: "Spatial AI Translator" — 공간 데이터 변환 도구

### 컨셉
서로 다른 SLAM 시스템/포맷 간의 맵 데이터를 변환하는 도구. 마치 번역가가 언어를 변환하듯, SLAM 데이터를 변환.

### 왜 필요한가
- 회사마다 다른 SLAM 시스템 사용 (ORB-SLAM, LIO-SAM, Cartographer 등)
- 시스템 교체 시 기존 맵 데이터 재활용 불가
- 멀티 로봇 환경에서 서로 다른 맵 포맷 통합 필요

### 왜 Sam만 할 수 있나
- SLAM 내부 구조 이해 + Localization(변환/적응) 마인드셋
- "번역"이라는 개념을 기술 영역에 적용하는 독특한 관점

### 수익 모델
- SaaS: $199~$999/월
- Enterprise: 커스텀 통합 $5,000~$20,000

### 현실 점검
- **기술 난이도**: 높음 (6~12개월 개발)
- **시장**: 니치하지만 명확한 페인 포인트
- **경쟁**: 거의 없음
- **점수**: 독특함 ◎◎◎ / 즉시 수익 × / 기술 가치 ◎◎◎

## 아이디어 4: "SLAM Data Marketplace" — SLAM 데이터셋 마켓플레이스

### 컨셉
SLAM 알고리즘 개발/테스트를 위한 고품질 데이터셋을 거래하는 마켓플레이스.

### 왜 필요한가
- SLAM 연구/개발에 양질의 데이터셋이 필수
- KITTI, TUM, EuRoC 등 공개 데이터셋은 있지만 특정 환경(창고, 농장, 실내 등) 데이터 부족
- 스타트업들이 자체 데이터 수집에 비용과 시간 소모

### 수익 모델
- 데이터셋 판매 수수료: 15~25%
- 프리미엄 데이터셋 직접 제작/판매: $500~$5,000/set
- 데이터 어노테이션 서비스: $2,000~$10,000/프로젝트

### 현실 점검
- **시장 존재**: 일부 학술 데이터셋은 있지만 상업적 마켓플레이스는 없음
- **초기 문제**: 닭과 달걀 (판매자와 구매자 동시 확보)
- **차별화**: SLAM 도메인 전문가가 큐레이션
- **점수**: 독특함 ◎◎ / 즉시 수익 △ / 네트워크 효과 ◎

## 아이디어 5: "Robot Localization Testing" — 로봇 현지 환경 테스트 서비스

### 컨셉
해외 로봇 회사가 한국 시장 진출 시, 한국 환경(아파트, 오피스, 지하 주차장 등)에서의 SLAM 성능 테스트를 대행.

### 왜 필요한가
- 로봇의 SLAM은 환경에 따라 성능이 크게 달라짐
- 미국에서 잘 작동하는 로봇이 한국 아파트에서 실패할 수 있음
- 해외 회사는 한국 환경에서 테스트할 인프라가 없음

### 왜 Sam만 할 수 있나
- SLAM 기술 이해 + 한국 환경 접근성 + Localization 마인드
- "기술 + 현지" 테스트를 할 수 있는 사람

### 수익 모델
- 테스트 리포트: $3,000~$10,000/건
- 지속적 테스트 계약: $2,000~$5,000/월
- 컨설팅 (환경 적응 가이드): $200~$300/hr

### 현실 점검
- **시장**: 한국 진출 로봇 회사 증가 (Bear Robotics 등 역방향도)
- **진입 장벽 낮음**: 장비와 환경 접근성만 있으면 시작 가능
- **스케일 제한**: 물리적 테스트 → 시간 교환
- **점수**: 독특함 ◎◎ / 즉시 수익 ○ / 스케일 △

## 아이디어 6: "AI-Powered SLAM Tuning" — SLAM 파라미터 자동 튜닝 도구

### 컨셉
SLAM 알고리즘의 수십 개 파라미터를 AI(강화학습/베이지안 최적화)로 자동 튜닝하는 도구.

### 왜 필요한가
- SLAM 파라미터 튜닝은 "블랙 아트" — 경험 많은 엔지니어만 가능
- 새로운 환경/센서마다 재튜닝 필요
- 자동화하면 SLAM 배포 시간을 10배 단축

### 왜 Sam만 할 수 있나
- 5년간 직접 튜닝해본 경험 (어떤 파라미터가 중요한지 앎)
- AI/ML 지식으로 자동 최적화 구현 가능
- "SLAM + AI" 교차점의 전형적 제품

### 수익 모델
- SaaS: $299~$1,999/월
- 컨설팅 (도구 포함): $5,000~$20,000/프로젝트

### 현실 점검
- **기술 난이도**: 중~상 (PoC 3개월, 상용화 6~12개월)
- **시장**: SLAM을 사용하는 모든 회사 = 명확한 타겟
- **경쟁**: 학술 논문은 있지만 상용 제품은 거의 없음
- **점수**: 독특함 ◎◎◎ / 기술 가치 ◎◎◎ / 수익 잠재력 ◎◎

## 아이디어 7: "Build in Public: From Zero to SLAM Freelancer" — 빌드인퍼블릭

### 컨셉
"매출 0, 인지도 0에서 SLAM 프리랜서로 월 $10K 달성까지" 전 과정을 실시간 공개.

### 왜 독특한가
- SLAM 엔지니어의 프리랜서 전환기를 공개하는 사람이 없음
- 기술 콘텐츠 + 비즈니스 여정 + 개인 스토리 = 강력한 내러티브
- 트위터/유튜브에서 "build in public" 트렌드와 결합

### 수익 모델
- **직접 수익**: 팔로워 → 스폰서십, 광고, 어필리에이트
- **간접 수익**: 인지도 → 프리랜싱 단가 상승 + 교육 비즈니스 론칭
- **네트워크 효과**: 로보틱스 업계 사람들이 자연스럽게 연결

### 실행 방법
```
매주 공개:
  - 이번 주 수입 (투명하게)
  - 수주한/비딩한 프로젝트
  - 배운 기술적/비즈니스적 교훈
  - 실패/거절 경험

플랫폼:
  - Twitter/X (영어): 주 3-5회 포스트
  - 유튜브 (영어): 월 2-4회 영상
  - 블로그: 주 1회 상세 글
```

### 현실 점검
- **비용**: 0 (시간만 투자)
- **리스크**: 개인 정보 노출, 실패 공개의 부담
- **장점**: 프리랜싱하면서 동시에 자산(오디언스) 축적 = **완벽한 병행 활동**
- **점수**: 독특함 ◎◎ / 즉시 실행 ◎◎◎ / 장기 가치 ◎◎◎

## Round 8 결론

### 핵심 발견
기존 전략(SLAM 프리랜싱 + 교육)에 **즉시 추가할 수 있는 비정형 아이디어**:

1. **Build in Public (아이디어 7)** — 프리랜싱 여정을 공개하면서 오디언스 구축. 추가 비용 0, 시너지 최대.
2. **SLAM Interview Prep (아이디어 2)** — 교육 비즈니스의 첫 번째 상품으로 적합. 니치 독점 가능.
3. **AI SLAM Tuning Tool (아이디어 6)** — 장기적으로 가장 가치 있는 SaaS 기회. Year 2부터 사이드 프로젝트로.

### 수정된 전략 조합
```
기존: SLAM 프리랜싱(현금) + SLAM 교육(자산)
  ↓
추가: + Build in Public(오디언스) + Interview Prep(첫 상품) + AI SLAM Tuning(장기 SaaS)
```

> **Build in Public은 비용 0으로 프리랜싱과 교육 비즈니스 모두를 가속하는 부스터다.**
> 오늘부터 트위터 계정 만들고 첫 포스트를 올려라.
