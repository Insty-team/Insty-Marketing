## 각 시나리오의 "이게 되려면"

### 보수적이 되려면 (확률 30%)
- SaaS MVP를 12개월 내 출시
- PMF를 찾을 때까지 포기하지 않음 (2~3번 pivot 가능)
- 월 5% 성장을 10년간 유지
- 정규직 병행하며 번아웃하지 않음

### 기본이 되려면 (확률 15%)
- 6개월 내 PMF 달성 (첫 시도 또는 빠른 pivot)
- 글로벌 시장 타겟 (한국만으로는 부족)
- 멀티 프로덕트 전략 성공 (2~3개 제품)
- 월 8% 성장을 5년간 유지
- 부동산 투자 타이밍 적절

### 공격적이 되려면 (확률 5%)
- 즉시 PMF (이미 시장이 있는 곳에 뛰어듦)
- 바이럴 성장 (유저가 유저를 데려옴)
- 투자 유치 성공 (SLAM+AI 스토리)
- 월 12% 성장을 3년간 유지 (Top 1% SaaS)

## 마일스톤별 해야 할 것

### 마일스톤 1: MRR 100만원 (첫 매출)

**기간:** 0~6개월
**행동:**
1. MVP 출시 (4~8주)
2. 무료 사용자 50명 확보
3. 유료 전환 10명 (전환율 20%)
4. 고객 단가 월 10만원 × 10명 = 100만원

### 마일스톤 2: MRR 1,000만원 (PMF)

**기간:** 6~18개월
**행동:**
1. 고객 피드백 기반 제품 개선
2. 마케팅 채널 확보 (SEO, 커뮤니티, Product Hunt)
3. 가격 최적화
4. 두 번째 제품 시작

### 마일스톤 3: MRR 5,000만원 (스케일)

**기간:** 18~36개월
**행동:**
1. 풀타임 전환 (퇴사 후)
2. 팀 구성 (프리랜서 1~2명)
3. 글로벌 확장 (영어권)
4. 엔터프라이즈 고객 확보

### 마일스톤 4: ARR 10억 (100억 밸류)

**기간:** 36~72개월
**행동:**
1. 팀 5~10명
2. 마켓플레이스 론칭 (네트워크 효과)
3. 투자 유치 또는 매각 검토
4. 자산 다각화 (부동산, ETF)
