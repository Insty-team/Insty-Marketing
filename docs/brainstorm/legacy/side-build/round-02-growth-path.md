## 1. 사업 모델별 성장 곡선 비교

### A. 프리랜싱/대행 (노동 = 돈)

```
월수입
2000만 ┤                           ← 시간 한계 벽
1500만 ┤                     ●●●●●
1000만 ┤               ●●●●●
 500만 ┤         ●●●●●
 200만 ┤   ●●●●●
     0 ┤●●●
       └─────────────────────────────
        1  3  6  9  12  18  24  (개월)
```

| 항목 | 값 |
|------|-----|
| 첫 수입까지 | 1-2주 (빠름) |
| 월 200만 도달 | 1-3개월 |
| 월 600만 도달 | 3-6개월 |
| 월 1000만 도달 | 6-12개월 |
| 천장 | **~1500만** (사이드 시간 한계) |
| Sam 시간 투자 | 주 18시간 × 시급 5-15만원 = 월 360-1,080만원 |
| 장점 | 즉시 수입, 예측 가능, 스킬 활용 |
| 단점 | 시간=돈 구조, 정규직 병행 시 시간 부족, 번아웃 |

**Sam에게 맞는 프리랜싱 형태**:
- AI 에이전트 자동화 구축 (시급 10-15만원)
- Claude Code 활용 풀스택 개발 (시급 8-12만원)
- **위험**: 정규직 9-6 + 프리랜싱 = 이중 노동. 지속 불가능

## 3. 최적 성장 경로: 하이브리드 전략

Sam에게 **순수 하나의 모델만 추천하면 안 된다**. 이유:
- SaaS만 하면 초기 6개월 수입 0 → 동기부여 붕괴
- 프리랜싱만 하면 시간=돈 함정 → 100억 불가능
- 디지털 상품만 하면 불규칙 수입 → 불안

### 추천 경로: "디지털 상품으로 시작 → SaaS로 전환"

#### Phase A: 디지털 상품 (월 1-4)
**목표**: 첫 수입 확인, 시장 감 익히기

| 월 | 활동 | 예상 수입 |
|----|------|---------|
| 1월 | AI 자동화 템플릿/가이드 제작 + Gumroad 출시 | 10-30만원 |
| 2월 | 2-3개 추가 상품 + 트위터/유튜브 마케팅 시작 | 30-80만원 |
| 3월 | 상품 포트폴리오 5-7개 + SEO 트래픽 | 50-150만원 |
| 4월 | 베스트셀러 상품 발견 + 업셀 구조 | 100-200만원 |

**장점**: 빠른 수입으로 동기부여 유지, 시장 검증, 청중 확보 시작

#### Phase B: 마이크로 SaaS 빌드 (월 3-8)
**목표**: MRR 구조 시작, 디지털 상품은 자동 판매 유지

| 월 | 활동 | 예상 수입 (디지털 상품 + SaaS) |
|----|------|------|
| 5월 | SaaS MVP 론칭 + 디지털 상품 유지 | 200만 + 10만 = 210만 |
| 6월 | SaaS 개선 + 유료 전환 시작 | 200만 + 30만 = 230만 |
| 8월 | SaaS PMF 확인 또는 피봇 | 200만 + 100만 = 300만 |

#### Phase C: SaaS 집중 성장 (월 9-24)
**목표**: MRR 복리 성장, 디지털 상품은 패시브

| 월 | SaaS MRR | 디지털 상품 | 합계 |
|----|---------|----------|------|
| 12개월 | 200만 | 200만 | 400만 (=정규직 수준) |
| 18개월 | 500만 | 200만 | 700만 |
| 24개월 | 1,000만 | 200만 | 1,200만 |

## 4. 구체적 성장 시나리오 (월별)

### 목표 대비 현실적 경로

| 목표 | 달성 시점 | 구성 |
|------|---------|------|
| 월 600만 (정규직+사이드) | **6-9개월** | 445(정규직) + 155(디지털 상품) |
| 월 800만 | **12-15개월** | 445 + 200(상품) + 155(SaaS) |
| 월 1,000만 | **15-20개월** | 445 + 200(상품) + 355(SaaS) |
| 월 2,000만 | **24-36개월** | 445 + 200(상품) + 1,355(SaaS) |

> **주의**: 이건 **정규직 수입 포함** 기준이다. 사이드만으로 월 2,000만원은 24-36개월 소요.

### 왜 이 경로가 Sam에게 맞는가

1. **동기부여**: 첫 달부터 소액이라도 수입이 보임 (디지털 상품)
2. **점진적 성장**: 매월 숫자가 올라감
3. **탈출 속도**: SaaS MRR이 정규직 수입을 넘으면 퇴사 옵션 생김
4. **리스크 최소**: 정규직 안전망 + 디지털 상품 = 실패해도 손해는 시간뿐
5. **100억 경로**: SaaS가 커지면 기업 가치 발생 (Round 8에서 상세)

## 6. Round 2 결론

### Sam의 최적 전략: "빠른 캐시플로 + 장기 자산 빌딩"

```
[디지털 상품] ──→ 즉시 수입 + 시장 학습 + 청중 구축
       ↓
[마이크로 SaaS] ──→ MRR 복리 성장 + 기업 가치
       ↓
[멀티 프로덕트] ──→ 100억 자산 경로 (Round 8)
```

- **하지 말 것**: 프리랜싱 (시간=돈 함정), 뉴스레터만 (너무 느림)
- **해야 할 것**: 디지털 상품으로 첫 수입 → SaaS로 전환
- **핵심 원칙**: 매월 숫자가 올라가야 한다. 정체되면 피봇
