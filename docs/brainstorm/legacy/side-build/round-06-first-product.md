## 상위 3개 후보 상세 분석

### 후보 1: HookGen — 숏폼 훅 생성기

**제품 정의**: 숏폼 영상의 첫 3초 훅을 AI로 생성하는 도구. 바이럴 훅 패턴 DB + 니치별 맞춤 생성.

| 평가 항목 | 점수 | 근거 |
|----------|------|------|
| 4-8주 MVP | ★★★★★ | 3-4주면 충분. 훅 DB + AI 생성 = 기술적으로 단순 |
| 첫 달 수입 | ★★★★★ | Gumroad에서 훅 DB를 디지털 상품으로 즉시 판매 가능 |
| 장기 MRR | ★★★ | $9-29/월 단가. 볼륨 필요. API로 확장 가능 |
| Sam 스킬 | ★★★★ | 콘텐츠 파이프라인 스킬 활용 |
| **총점** | **85/100** | |

**시장 검증**:
- 숏폼 비디오 = 전체 인터넷 트래픽 82% (2026)
- YouTube Shorts 일일 700억 조회
- TikTok 20억+ 사용자
- 기존 경쟁자: Virlo (분석 중심), Viral.app (트래킹 중심) — **훅 생성 전용 도구는 거의 없음**

**구체적 실행 계획**:
```
주 1-2: 바이럴 훅 500개 수집 + 패턴 분류 (10개 니치)
주 2: Gumroad에 "Viral Hooks Database" $19로 출시 (디지털 상품)
주 3-4: AI 훅 생성기 MVP 빌드 (Next.js + Claude API)
주 5: SaaS 버전 론칭 ($9/월 or $29/월)
주 6-8: 피드백 → 개선 → 트위터/틱톡 마케팅
```

**월별 수입 예상**:
| 월 | 디지털 상품 | SaaS | 합계 |
|----|-----------|------|------|
| 1 | 15만 | 0 | 15만 |
| 2 | 20만 | 5만 | 25만 |
| 3 | 25만 | 15만 | 40만 |
| 6 | 30만 | 50만 | 80만 |
| 12 | 30만 | 150만 | 180만 |

### 후보 2: NicheNewsAI — 뉴스레터 자동화 도구

**제품 정의**: 특정 키워드/산업 뉴스를 AI가 자동 수집 → 큐레이션 → 뉴스레터 초안 생성. 크리에이터의 뉴스레터 작성 시간 80% 절감.

| 평가 항목 | 점수 | 근거 |
|----------|------|------|
| 4-8주 MVP | ★★★★ | 4-6주. RSS 수집 + AI 요약 + 마크다운 출력 |
| 첫 달 수입 | ★★★ | 뉴스레터 크리에이터 대상 = 타겟 좁음 |
| 장기 MRR | ★★★★ | $19-49/월. 뉴스레터 시장 138% 성장 |
| Sam 스킬 | ★★★★★ | 콘텐츠 파이프라인 = 정확히 이 도구 |
| **총점** | **78/100** | |

**시장 검증**:
- Substack 유료 구독자 5M+ (2025)
- 뉴스레터 유료 구독 138% 성장 (2024→2025)
- 경쟁자: Curateflow (Beehiiv 전용, 초기 단계) — 시장 초기

**구체적 실행 계획**:
```
주 1-2: 직접 AI 뉴스레터 운영 시작 (dogfooding + 마케팅)
주 3-6: 도구 MVP 빌드 (소스 수집 + AI 큐레이션 + 출력)
주 7-8: 베타 사용자 10명 모집 + 피드백
```

**문제점**: 첫 달 수입이 느림. Dogfooding으로 뉴스레터 자체 수익화까지 6개월+.

### 후보 3: ShipKit — AI SaaS 보일러플레이트

**제품 정의**: AI SaaS를 빠르게 만들기 위한 풀스택 보일러플레이트 키트. Next.js + LLM SDK + Stripe + Auth + 대시보드.

| 평가 항목 | 점수 | 근거 |
|----------|------|------|
| 4-8주 MVP | ★★★★ | 4-6주. 이미 만들어본 스택 기반 |
| 첫 달 수입 | ★★★★★ | 원타임 $99-199. Marc Lou ShipFast 첫 달 $40K |
| 장기 MRR | ★★★ | 원타임 + 업데이트 구독 하이브리드 |
| Sam 스킬 | ★★★★★ | Claude Code + 풀스택 = 직접 사용하는 스택 |
| **총점** | **82/100** | |

**시장 검증**:
- ShipFast: 첫 달 $40K, 현재 $100K+/월
- Shipped, LaunchFast 등 경쟁자 다수
- 하지만 "AI 에이전트 전용" 보일러플레이트는 거의 없음

**문제점**: ShipFast 브랜드 파워가 너무 강함. Marc Lou 팔로워 vs Sam 0명.

## 최종 추천: HookGen

### 왜 HookGen인가?

1. **디지털 상품 → SaaS 전환이 자연스럽다**
   - Week 2: 훅 DB를 Gumroad에서 $19에 판매 (디지털 상품)
   - Week 5: AI 훅 생성기 SaaS 론칭 ($9-29/월)
   - Round 2의 "디지털 상품으로 시작 → SaaS로 전환" 전략과 정확히 일치

2. **첫 수입이 빠르다**
   - 훅 DB는 만드는 데 1주일. 바로 판매 가능
   - Sam의 "동기부여를 위해 수입이 보여야 한다" 조건 충족

3. **시장이 거대하고 경쟁이 약하다**
   - 숏폼 크리에이터 수천만 명
   - "훅 생성"에만 집중하는 도구 = 블루오션
   - 기존 경쟁자(Virlo, Viral.app)는 분석/트래킹 중심

4. **기술적으로 간단하다**
   - 훅 DB + AI 생성 = Claude API + 간단한 웹앱
   - 주 18시간으로 4주 내 MVP 충분

5. **확장 경로가 명확하다**
   - HookGen → ContentPipe (콘텐츠 리퍼포징) → 크리에이터 도구 플랫폼
   - API 서비스로 B2B 확장 가능

### 왜 다른 것이 아닌가?

- **NicheNewsAI**: 훌륭하지만 첫 수입이 너무 느림. 두 번째 제품으로 적합
- **ShipKit**: 수입은 빠르지만 경쟁 치열 + 원타임 = MRR 아님. 세 번째 또는 병행 상품으로
