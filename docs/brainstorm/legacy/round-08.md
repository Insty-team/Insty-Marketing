# 8차 고민 — 오늘부터의 90일

> 이전 차수에서 발전/수정한 점: Round 7에서 시나리오 B(솔로프리너 중심)를 기본 경로로 선택했다. 고단가 세팅(₩50~150만) + 유지보수 구독(₩30만/월) + 디지털 상품 + Build in Public 콘텐츠가 핵심 수익원. 이번에는 이 경로의 첫 90일을 주 단위로 설계한다.

## 핵심 질문
시나리오 B의 경로를 실행하기 위해, 오늘부터 90일 동안 정확히 뭘 해야 하는가?

## 생각 전개

### 1. 90일의 3단계

```
Phase A (Week 1~4):  "무기 장전" — 오퍼 정의, 첫 고객, 콘텐츠 시작
Phase B (Week 5~8):  "첫 증명" — 유료 세팅 3건, 사례 구축, 루틴 확립
Phase C (Week 9~13): "반복 수익" — 유지보수 전환, 디지털 상품 출시, 커뮤니티 씨앗
```

### 6. 90일 후 성공/실패 판단 기준

**계속 진행 조건 (하나라도 미달이면 피봇 고려):**
```
✅ 유료 고객 3건 이상 (세팅)
✅ 월 수입 ₩200만 이상 (세팅 + 구독 + 디지털 합산)
✅ 유지보수 구독자 3명 이상 (반복 수익 존재)
✅ 인바운드 문의 주 1건 이상 (콘텐츠가 작동 중)
```

**피봇 필요 신호:**
```
❌ DM 400건 보냈는데 유료 계약 0건 → 오퍼/타겟/메시지 중 하나가 틀림
❌ 세팅은 되는데 유지보수 전환 0건 → 가치가 부족하거나 가격이 높음
❌ 콘텐츠 80개인데 팔로워 100명 미만 → 콘텐츠 전략이 틀림
```

### 7. 비상 계획 (Plan B)

90일 후 목표 미달 시:
1. **타겟 피봇**: 솔로프리너 → 소상공인 (또는 그 반대)
2. **가격 피봇**: ₩50만 → ₩20만 (or 성과 보수 모델)
3. **서비스 피봇**: 파이프라인 구축 → 단순 자동화 세팅
4. **채널 피봇**: DM → 커뮤니티 침투 → 숏폼 중심

**절대 포기하지 않는 것: Build in Public 콘텐츠 (이것은 모든 시나리오에서 자산)**

## 이전 차수 오류 수정

Round 7에서 시나리오 B의 Month 1~3 월 수입을 ₩67만으로 잡았는데, 이번 주 단위 계획으로 보면 Week 4에 첫 유료 계약(₩50~100만)이 발생하므로 M1은 ₩0~50만, M2는 ₩100~150만, M3는 ₩200~300만으로 **점진적 증가**가 더 현실적이다. 평균 ₩67만은 M1의 ₩0이 끌어내리는 것이므로, "3개월 평균"보다 "3개월 말 런레이트"로 보는 것이 더 의미 있다.
