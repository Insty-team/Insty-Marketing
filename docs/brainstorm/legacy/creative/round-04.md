## 상위 시너지 조합 분석

### 조합 1: ① SLAM 프리랜싱 × E. 교육 비즈니스 ⭐ 최강 조합

```
[단기] SLAM 프리랜싱으로 월 ₩650만~₩1,300만 확보
  ↓ 동시에
[장기] 프리랜싱 과정에서 겪는 문제/해결을 콘텐츠로 제작
  ↓ 축적
[자산] "실전 SLAM 엔지니어의 프리랜싱 + 기술 노하우" 채널/코스
```

**시너지 포인트:**
- 프리랜싱 프로젝트 자체가 콘텐츠 소재 (NDA 주의, 일반화해서 공유)
- "현직 SLAM 프리랜서가 가르치는" → 이론 교수보다 신뢰도 높음
- 돈 벌면서 동시에 콘텐츠 자산 축적
- 교육 브랜드가 프리랜싱 단가를 올려줌 (선순환)

**3년 시뮬레이션:**
```
Year 1:
  - SLAM 프리랜싱: ₩1억 (세후 ~₩8,000만)
  - 유튜브 시작 + 첫 코스: ₩500만
  - 합계: ~₩8,500만

Year 2:
  - SLAM 프리랜싱 (단가 인상): ₩1.2억 (세후 ~₩9,500만)
  - 유튜브 + 코스 성장: ₩3,000만
  - 합계: ~₩1.25억

Year 3:
  - SLAM 프리랜싱 축소 (주 15시간): ₩6,000만
  - 교육 비즈니스 폭발: ₩8,000만
  - 합계: ~₩1.4억

3년 합계: ~₩3.5억 (생활비 차감 후 ~₩2.8억)
```

**평가:** 3.1억에 근접. 프리랜싱 시간을 줄이면서 교육이 대체하는 이상적 전환.

### 조합 2: ① SLAM 프리랜싱 × D. SLAM × Localization IP화

```
[단기] SLAM 프리랜싱으로 현금 + 다양한 프로젝트 경험
  ↓ 동시에
[장기] 로보틱스 제품 글로벌 배포 경험을 방법론/프레임워크로 체계화
  ↓ 축적
[자산] "로보틱스 제품 현지화 전문가" IP (컨설팅 → 워크숍 → SaaS)
```

**시너지 포인트:**
- 프리랜싱하면서 다양한 로보틱스 회사의 글로벌 진출 니즈를 직접 관찰
- 클라이언트에게 "현지화도 도와드릴까요?"로 업셀
- 사례 축적 → 방법론 정리 → IP화
- **최고의 독특성** — 이 교차점의 전문가가 세상에 거의 없음

**3년 시뮬레이션:**
```
Year 1:
  - SLAM 프리랜싱: ₩1억
  - 현지화 컨설팅 2건: ₩2,000만
  - 합계: ~₩1.2억 (세후 ~₩9,500만)

Year 2:
  - SLAM + 현지화 번들 컨설팅: ₩1.5억
  - 워크숍/코스 시작: ₩2,000만
  - 합계: ~₩1.7억 (세후 ~₩1.3억)

Year 3:
  - 프리미엄 컨설팅 + IP 판매: ₩2억
  - 합계: ~₩2억 (세후 ~₩1.5억)

3년 합계: ~₩3.7억 (생활비 차감 후 ~₩3억)
```

**평가:** 3.1억 달성 가능. 하지만 "로보틱스 현지화" 시장 크기가 핵심 변수.

### 조합 3: ⑤ 기술 자문 × D. SLAM × L10n IP × E. 교육

```
[단기] 3~4개 스타트업에 기술 자문 (주 12~20시간, 월 ₩400만~₩1,200만)
  ↓ 남은 시간으로
[장기] 자문 경험을 교육 콘텐츠와 IP로 전환
  ↓ 축적
[자산] 업계 네트워크 + 교육 브랜드 + 방법론 IP
```

**시너지 포인트:**
- 자문은 시간 효율 최고 (주 4시간/회사 × 3~4개)
- 다양한 스타트업의 기술 스택을 볼 수 있음 → 콘텐츠 소재 풍부
- 스톡옵션 = 잭팟 가능성 (3년 내 엑싯 시 보너스)
- 네트워크 확장 → 미래 고객 파이프라인

**약점:** 초기 네트워크 부족으로 자문 기회 확보가 관건.

### 조합 4: ② AI 에이전시 × B. Agent-in-a-Box

```
[단기] AI 에이전트 구축 프로젝트 수주 (한국 + 글로벌)
  ↓ 반복되는 패턴을 발견
[장기] 패턴을 템플릿화 → "Agent-in-a-Box" 상품으로 판매
  ↓ 축적
[자산] 산업별 AI 에이전트 템플릿 라이브러리
```

**시너지 포인트:**
- 커스텀 프로젝트 → 공통 요소 추출 → 템플릿화 (자연스러운 전환)
- 클라이언트 피드백으로 제품 개선
- 에이전시 레퍼런스 = 템플릿 신뢰도

**약점:** AI 에이전트 시장 경쟁 치열, Sam의 진짜 차별점(SLAM)을 활용 못함.

### Tier 1 (최강 추천)

#### 🏆 "SLAM 프리랜서 + 교육 빌더" (조합 1)

```
저녁/주말: 유튜브 콘텐츠 + 코스 제작 (2~3시간)
```
- **단기**: 프리랜싱으로 생활비 + 저축 (월 ₩650만~₩1,300만)
- **장기**: 교육 비즈니스가 프리랜싱을 대체 (Year 2~3)
- **복리**: 콘텐츠 축적 + 구독자 성장 + 브랜드 가치
- **출구**: "SLAM/AI 교육의 Think Autonomous" 되기

### Tier 2 (차선)

#### "SLAM 프리랜서 + SLAM×L10n 전문가" (조합 2)
- 프리랜싱하면서 현지화 니즈 발견 → IP화
- 더 독특하지만 시장 크기 리스크

### Tier 3 (대안)

#### "기술 자문 + 교육 + IP" (조합 3)
- 시간 효율 최고, 하지만 네트워크 선행 필요

## 핵심 인사이트

> **"돈 버는 활동"과 "자산 쌓는 활동"이 같은 도메인에 있어야 시너지가 극대화된다.**
>
> Sam의 경우: SLAM 프리랜싱(돈) + SLAM 교육(자산) = 같은 도메인 = 최대 시너지
>
> 반면: SLAM 프리랜싱(돈) + Insty 플랫폼(자산) = 다른 도메인 = 에너지 분산

이것이 기존 Insty 비전의 가장 큰 문제였다. **돈 버는 활동과 자산 쌓는 활동이 서로 다른 도메인이라 시너지가 없었다.**
