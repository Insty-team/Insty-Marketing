# Round 7: 한국 vs 글로벌 — 듀얼 트랙 전략

> Round 1~6 핵심: SLAM 프리랜싱(단기) + SLAM 특화 교육(장기)이 최적 조합.
> 이번 라운드: 한국 시장과 글로벌 시장, 어디에서 어떻게 활동하면 단가가 극대화되는가?

---

## 1. 글로벌 vs 한국: 단가 비교

### 프리랜서 시급 비교 (2025~2026)

| 스킬 | 미국/서유럽 | 한국 | 차이 |
|------|----------|------|------|
| 일반 소프트웨어 개발 | $60~$120/hr | ₩4만~8만/hr ($30~$60) | **2배** |
| AI/ML 엔지니어 | $100~$200/hr | ₩7만~15만/hr ($50~$110) | **1.5~2배** |
| SLAM/로보틱스 | $100~$300/hr | ₩10만~20만/hr ($75~$150) | **1.5~2배** |
| AI 에이전트 개발 | $75~$300/hr | ₩7만~20만/hr ($50~$150) | **1.5~2배** |

### 핵심 수치
> **같은 시간을 쓰면 글로벌 시장에서 1.5~2배 더 번다.**
> 서울 생활비(월 200만원)를 고려하면, 글로벌 단가로 서울에 사는 것은 **지오 아비트라지**의 전형적 사례다.

---

## 2. 지오 아비트라지 (Geo-Arbitrage) 분석

### Sam의 지오 아비트라지 시나리오

```
글로벌 단가로 프리랜싱: $100/hr × 30hr/week × 4주 = $12,000/월 (약 ₩1,600만)
서울 생활비: ₩200만/월
────────────
순수익: ₩1,400만/월 (약 $10,400)
저축 가능액: 연 ₩1.68억

vs 한국 단가로 프리랜싱: ₩10만/hr × 30hr/week × 4주 = ₩1,200만/월
순수익: ₩1,000만/월
저축 가능액: 연 ₩1.2억
```

> **글로벌 단가 vs 한국 단가의 차이: 연 ₩4,800만 (3년이면 ₩1.44억)**
> 이 차이는 3.1억 목표 달성 여부를 가를 수 있다.

## 3. 글로벌 시장 진입 전략

### 플랫폼별 특성

| 플랫폼 | 특성 | SLAM 적합도 | 예상 시급 | 수수료 |
|--------|------|-----------|----------|--------|
| **Upwork** | 가장 큰 프리랜서 마켓 | ★★★★ (SLAM 프로젝트 실제 존재) | $80~$150 | 10% |
| **Toptal** | 상위 3% 선별 | ★★★★★ (고급 엔지니어 매칭) | $100~$200 | 0% (클라이언트 부담) |
| **Arc.dev** | 리모트 개발자 특화 | ★★★ | $75~$150 | 낮음 |
| **LinkedIn** | 직접 네트워킹 | ★★★★ | $100~$300 (직접 협상) | 0% |
| **개인 웹사이트** | 브랜드 빌딩 | 장기적 최고 | $150~$300+ | 0% |

### 진입 우선순위

```
Week 1-2: Upwork 프로필 최적화 + 첫 프로젝트 비딩
  → 가장 빠른 진입, SLAM 프로젝트 실제 존재
  → 초기 3~5건은 단가 양보해서 리뷰 확보

Week 3-4: Toptal 심사 신청
  → 통과하면 고단가 자동 매칭
  → SLAM 5년 + AI 경험이면 통과 가능성 높음

Month 2-3: LinkedIn 포지셔닝
  → "SLAM + AI 프리랜서 전문가" 프로필
  → 주 1회 기술 포스트 (영문)
  → 로보틱스 스타트업 창업자/CTO와 네트워킹

Month 4-6: 개인 웹사이트 + 직접 고객
  → 플랫폼 수수료 0%
  → 브랜드 기반 인바운드 리드
```

## 4. 한국 시장의 기회

### 한국에서만 할 수 있는 것

#### 기회 1: 한국 로보틱스 스타트업 생태계
- 42dot, Autonomous a2z, NaverLabs, Bear Robotics 등
- SLAM 인력 만성 부족 → 프리랜서/자문 수요
- 대면 미팅 가능 → 신뢰 구축 빠름

#### 기회 2: 한국 정부 R&D 프로젝트
- 자율주행, 로봇, AI 관련 정부 과제 활발
- 프리랜서/PM으로 참여 가능 (연 ₩3,000만~₩8,000만)

#### 기회 3: 한국어 SLAM/AI 교육 시장
- 한국어 SLAM 교육 콘텐츠는 거의 전무
- 한국 로보틱스 엔지니어들의 학습 수요 존재
- 소규모지만 독점 가능

### 한국 시장의 한계
- 프리랜서 단가가 글로벌 대비 낮음
- 프리랜서 문화가 상대적으로 약함 (정규직 선호)
- 대기업 중심 의사결정 → 개인 프리랜서 진입 어려움

## 5. 듀얼 트랙 전략: 최적 설계

### 전략: "글로벌로 벌고, 한국으로 네트워크 쌓고, 영어로 교육 자산 만든다"

```
┌─────────────────────────────────────────────┐
│              Sam의 듀얼 트랙                  │
├──────────────┬──────────────────────────────┤
│   현금 트랙    │        자산 트랙              │
│   (글로벌)     │      (글로벌 + 한국)          │
├──────────────┼──────────────────────────────┤
│ Upwork/Toptal │ 영어 유튜브/블로그             │
│ SLAM 프리랜싱  │ SLAM + AI 교육 코스           │
│ $100-200/hr   │ 한국 로보틱스 커뮤니티 네트워킹  │
│               │ 한국어 콘텐츠 (보조)            │
└──────────────┴──────────────────────────────┘
```

### 시간 배분 (주 50시간 기준)

| 활동 | 시간/주 | 시장 | 목적 |
|------|--------|------|------|
| SLAM 프리랜싱 | 30시간 | 글로벌 | 현금 |
| 영어 콘텐츠 제작 | 10시간 | 글로벌 | 장기 자산 |
| 한국 네트워킹 | 5시간 | 한국 | 네트워크/자문 기회 |
| 한국어 콘텐츠 (선택) | 5시간 | 한국 | 보조 수입 |

### 왜 영어 콘텐츠 우선인가

| 비교 항목 | 영어 콘텐츠 | 한국어 콘텐츠 |
|----------|-----------|-------------|
| 잠재 시장 크기 | 10억+ 영어권 | 5,000만 한국 |
| SLAM 관심 인구 | 수십만 명 | 수천 명 |
| CPM (유튜브) | $15~$30 | $3~$8 |
| 코스 결제 의향 | $100~$500 보편적 | 높은 가격 저항 |
| 경쟁 | 있지만 파이가 큼 | 적지만 파이도 작음 |

> **영어 콘텐츠의 수익 잠재력이 한국어의 5~10배.**
> 한국어는 네트워킹과 브랜딩 목적으로 보조적으로 활용.

## 6. 시간대 장점/단점

### 한국(KST)에서 글로벌 프리랜싱

**미국 클라이언트 (시차 14~17시간)**:
- 비동기 작업에 적합 (미국이 자는 동안 작업 → 아침에 결과 전달)
- 실시간 미팅 어려움 (한국 오전 = 미국 저녁, 가능하지만 불편)
- 해결법: 주 1~2회 미팅으로 제한, 나머지는 비동기

**유럽 클라이언트 (시차 7~9시간)**:
- 한국 오후 = 유럽 오전 → 겹치는 시간 3~4시간
- 더 수월한 협업 가능

**추천**: 초기에는 **유럽 클라이언트 우선** 타겟 → 시차 관리 용이

## 7. 3년 듀얼 트랙 로드맵

### Year 1: 기반 구축
```
월 1-3: Upwork/Toptal에서 SLAM 프리랜싱 시작
  → 월 ₩800만~₩1,300만 현금 확보
  → 유튜브 채널 개설, 주 1회 영어 콘텐츠

월 4-6: 프리랜싱 안정화 + 콘텐츠 성장
  → 단가 인상 ($100 → $120/hr)
  → 한국 로보틱스 밋업 참석, 네트워크 확장

월 7-12: 첫 유료 코스 런칭
  → 유튜브 구독자 2,000~5,000명
  → 첫 코스 ($199) 런칭, 50건 판매 목표

Year 1 예상 수입: ₩1.2억~₩1.8억
```

### Year 2: 가속
```
프리랜싱: $130~$150/hr, 주 25시간으로 축소
  → 월 ₩1,200만~₩1,500만
교육: 코스 2~3개, 코호트 프로그램 시작
  → 월 ₩300만~₩800만
한국: 자문 1~2건
  → 월 ₩200만~₩400만

Year 2 예상 수입: ₩1.8억~₩2.5억
```

### Year 3: 레버리지
```
프리랜싱: 주 15시간으로 축소 (고단가 선별적)
  → 월 ₩800만~₩1,000만
교육: 에버그린 코스 + 코호트 + 커뮤니티
  → 월 ₩800만~₩2,000만
자문/네트워크: 스톡옵션 포함 2~3건
  → 월 ₩300만~₩600만

Year 3 예상 수입: ₩2억~₩3억
```

### 3년 합계
```
보수적: ₩1.2억 + ₩1.8억 + ₩2억 = ₩5억 (생활비 차감 후 ₩4.3억) ✅
현실적: ₩1.5억 + ₩2억 + ₩2.5억 = ₩6억 (생활비 차감 후 ₩5.3억) ✅✅
```

> **3.1억 목표: 보수적 시나리오로도 충분히 달성 가능.**

## Round 7 결론

### 핵심 결정
1. **주 수입원은 글로벌 시장** — 단가 1.5~2배, 지오 아비트라지 극대화
2. **콘텐츠는 영어 우선** — 시장 크기 5~10배, CPM 3~5배
3. **한국은 네트워크/자문용** — 대면 미팅의 장점 활용, 자문 기회 확보
4. **유럽 클라이언트 우선 타겟** — 시차 관리 유리

### "한국 시장만 쓸 건가?"에 대한 답
> **절대 아니다.** 한국 시장만 타겟하면 3년 3.1억 달성이 빠듯해지고,
> 장기적으로 콘텐츠/교육의 스케일이 5~10배 줄어든다.
> 글로벌로 벌고, 한국에서 살고, 영어로 자산을 쌓는 것이 최적.
