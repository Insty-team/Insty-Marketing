# Round 2: 단기 현금 전략 5가지 — 가장 빠르게 월 500만원+

> Round 1에서 확인한 핵심: SLAM 5년은 과소평가된 금광, 교차 스킬 조합은 유니크, 시간당 가치를 최대화하는 것이 먼저.

## 전제 조건
- 현재 매출 0, 인지도 0, 레퍼런스 0
- 서울 거주, 생활비 월 200만원
- 1인 운영, 초기 투자 자본 거의 없음
- **목표: 월 500만원 이상 (연 6,000만원+)**
- 3년 내 3.1억 = 연평균 ~1.03억 필요

## 전략 1: SLAM/로보틱스 프리랜서 컨설팅 (글로벌 리모트)

### 개요
Upwork, Toptal, Arc.dev 등에서 SLAM/ROS2/자율주행 프리랜서로 활동. 글로벌 단가로 한국에서 생활.

### 구체적 숫자
- **시급**: $80~$150/hr (초기 $80, 실적 쌓으면 $120~$150)
- **주당 가용 시간**: 30시간 (풀타임 프리)
- **월 예상 수입**: $80 × 30hr × 4주 = **$9,600 (약 ₩1,300만)**
- **보수적 시나리오**: $60 × 20hr × 4주 = **$4,800 (약 ₩650만)**

### 현실성 분석
| 항목 | 평가 |
|------|------|
| 시작까지 걸리는 시간 | 2~4주 (프로필 세팅 + 첫 프로젝트 수주) |
| 실현 가능성 | **높음** — SLAM 5년 경력은 Upwork에서 즉시 경쟁력 |
| 천장 | $150~$200/hr (개인 브랜딩 후) |
| 스케일 가능성 | 낮음 (시간 교환) |
| 리스크 | 프로젝트 간 공백, 플랫폼 수수료 10~20% |

### 실제 사례
- Upwork에 "Senior Robotics Autonomy Engineer (LiDAR SLAM / ROS2 / PX4) – Remote Contract" 같은 프로젝트가 실제로 올라와 있음
- 로보틱스 프리랜서 Mat Sadowski: 5년간 리모트 로보틱스 컨설팅 성공적 운영

### 장원영 모델 대비
- 대체불가: ○ (SLAM 전문성)
- 한계비용 0: × (시간 교환)
- 복리: × (경험만 쌓임)
- **평가: 단기 현금 최적, 장기 전략 필요**

## 전략 2: AI 에이전트 구축 에이전시 (한국 중소기업 대상)

### 개요
한국 중소기업/1인 사업자에게 AI 에이전트(챗봇, 세일즈 자동화, 콘텐츠 자동화) 구축 서비스 제공. OpenClaw 경험을 레퍼런스로 활용.

### 구체적 숫자
- **프로젝트 단가**: ₩300만~₩1,000만 (복잡도에 따라)
- **월 리테이너**: ₩100만~₩300만/클라이언트
- **월 2건 수주 시**: ₩600만~₩2,000만
- **보수적 시나리오**: 월 1건 × ₩500만 = **₩500만**

### 현실성 분석
| 항목 | 평가 |
|------|------|
| 시작까지 걸리는 시간 | 4~8주 (포트폴리오 정리 + 영업) |
| 실현 가능성 | **중간** — 한국 시장에서 AI 에이전트 수요는 있지만, 인지도 0에서 영업이 과제 |
| 천장 | 월 ₩3,000만~₩5,000만 (클라이언트 증가 시) |
| 스케일 가능성 | 중간 (프로세스 표준화 → 하청 가능) |
| 리스크 | 영업 난이도, 클라이언트 교육 비용, 대금 회수 |

### 실제 사례
- 한국에서 대형 컨설팅사(Deloitte, PwC, EY)만 AI 에이전트 서비스를 제공 중
- **중소기업 대상 AI 에이전트 구축 에이전시는 블루오션에 가까움**
- Naver, Kakao가 2026년 agentic AI 전략 발표 → 시장 교육이 되고 있음

### 장원영 모델 대비
- 대체불가: △ (진입장벽 낮아지는 중)
- 한계비용 0: × (프로젝트마다 커스텀)
- 복리: △ (레퍼런스/케이스스터디 축적)
- **평가: 단기 가능하지만 차별화 전략 필수**

## 전략 3: 기술 콘텐츠 + 교육 (유튜브/강의)

### 개요
SLAM, AI 에이전트, 자동화 파이프라인에 대한 기술 콘텐츠를 영문으로 제작. 유튜브 + 유료 강의 (Udemy, own platform).

### 구체적 숫자
- **유튜브 CPM (영어, 기술 니치)**: $15~$30
- **목표 월간 조회수**: 100K → 월 $1,500~$3,000 (약 ₩200만~₩400만)
- **유료 강의 (Udemy/자체)**: $50~$200/건 × 100명/월 = $5,000~$20,000
- **도달 시나리오**: 6~12개월 후 월 ₩500만~₩1,000만 가능

### 현실성 분석
| 항목 | 평가 |
|------|------|
| 시작까지 걸리는 시간 | 6~12개월 (채널 성장 기간) |
| 실현 가능성 | **중간** — 니치 기술 콘텐츠는 충성도 높은 오디언스 확보 가능, 하지만 시간 소요 |
| 천장 | 무한대 (콘텐츠=자산) |
| 스케일 가능성 | **매우 높음** (한 번 만들면 계속 수익) |
| 리스크 | 초기 6개월 무수익, 유튜브 쇼츠 경험은 있지만 롱폼 다름 |

### 실제 사례
- Think Autonomous (thinkautonomous.ai): SLAM/자율주행 교육 플랫폼 운영
- Cyrill Stachniss: SLAM 유튜브 강의로 학계 인지도 구축
- 기술 유튜버들 월 $10K~$50K+ 수익 사례 다수

### 장원영 모델 대비
- 대체불가: ○ (SLAM 실무 경험 기반 콘텐츠)
- 한계비용 0: ○ (콘텐츠 복제)
- 복리: ○ (구독자/과정 축적)
- **평가: 장기 최적이지만 단기 현금 해결 불가. 병행 필수.**

## 전략 4: 한국 로보틱스/자율주행 회사 파트타임 계약

### 개요
한국의 자율주행/로보틱스 스타트업에 파트타임 또는 프로젝트 계약직으로 SLAM 엔지니어링 제공. 안정적 현금 확보.

### 구체적 숫자
- **계약직 월급**: ₩600만~₩1,000만 (시니어 SLAM)
- **파트타임 (주 3일)**: ₩400만~₩600만
- **풀타임 계약 (6개월)**: ₩800만~₩1,200만/월

### 현실성 분석
| 항목 | 평가 |
|------|------|
| 시작까지 걸리는 시간 | 2~4주 (이력서 + 면접) |
| 실현 가능성 | **매우 높음** — 한국 SLAM 인력 부족, 즉시 채용 가능 |
| 천장 | ₩1,200만/월 (풀타임 시니어) |

| 리스크 | 시간 구속, 사이드 프로젝트 제한 |

### 실제 사례

## 전략 5: AI + SLAM 기술 자문/어드바이저

### 개요
초기 스타트업에 기술 자문 (Technical Advisor)으로 참여. 시간 투자 최소화하면서 현금 + 스톡옵션 확보.

### 구체적 숫자
- **자문료**: 월 ₩100만~₩300만 (주 4~8시간)
- **스톡옵션**: 0.5%~2% (시리즈 A 이전 스타트업)
- **3~4개 회사 자문 시**: 월 ₩300만~₩1,200만 (주 12~24시간)

### 현실성 분석
| 항목 | 평가 |
|------|------|
| 시작까지 걸리는 시간 | 4~8주 (네트워킹 + 소개) |
| 실현 가능성 | **중간** — SLAM 전문성은 자문 가치 높지만, 네트워크 필요 |
| 천장 | 월 ₩1,200만 (자문료) + 스톡옵션 (잭팟 가능) |
| 스케일 가능성 | 중간 (시간 최소화 가능) |
| 리스크 | 스톡옵션 가치 불확실, 초기 네트워크 부족 |

### 장원영 모델 대비

- 복리: △ (스톡옵션이 복리 요소)
- **평가: 시간 대비 수입 효율 최고. 하지만 네트워크 구축이 선행 필요.**

## Round 2 결론

### 즉시 실행 가능한 "오늘부터" 전략
1. **전략 ①** (SLAM 글로벌 프리랜싱) — Upwork 프로필 만들고 바로 시작. 월 ₩650만~₩1,300만 가능.
2. **전략 ④** (파트타임 계약직) — 안정적 현금을 위한 보험. 주 3일 계약으로 ₩400만~₩600만.

### "내일 당장 시작할 수 있는가?" 테스트
- ① SLAM 프리랜싱: **YES** — Upwork 프로필 + SLAM 포트폴리오만 있으면 됨
- ② AI 에이전시: **MAYBE** — OpenClaw 케이스스터디 정리 필요 (1~2주)
- ③ 콘텐츠: **YES** — 유튜브 채널 개설은 오늘 가능, 하지만 수익까지 6개월+
- ④ 계약직: **YES** — 이력서 제출 내일 가능
- ⑤ 자문: **NO** — 네트워크 선행 필요

### 3년 3.1억 달성 시뮬레이션 (전략 ① 기준)
```
Year 1: $80/hr × 25hr/week × 48주 = $96,000 (약 ₩1.3억)
Year 2: $120/hr × 25hr/week × 48주 = $144,000 (약 ₩1.9억)
Year 3: $150/hr × 20hr/week × 48주 = $144,000 (약 ₩1.9억)
                                       ────────
3년 합계:                              약 ₩5.1억 (생활비 차감 후 ~₩3.9억)
```
> **SLAM 프리랜싱만으로도 3년 3.1억은 수학적으로 달성 가능하다.**
> 하지만 이건 "노동=돈" 모델이다. Round 3에서 장기 스케일 전략을 탐구한다.
