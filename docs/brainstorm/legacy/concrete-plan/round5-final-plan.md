## 1. 비전 한 문장

> **"월급만으로는 원하는 삶을 못 사는 직장인이, AI를 무기로 자기 자산을 만들어가는 과정을 기록하고, 같은 길을 걷는 사람들의 거점을 소유한다."**

## 2. 전략 한 단락

Sam은 SLAM 엔지니어라는 정규직을 유지하면서, 퇴근 후 주 18시간을 투자해 유튜브 쇼츠(메인) + 트위터/X(보조) + 뉴스레터(전환 퍼널)를 통해 "직장인이 AI로 사이드 사업 만드는 과정"을 리얼타임으로 기록한다. 6개월간 콘텐츠로 구독자 1,000~2,000명을 모은 후 Discord 기반 유료 커뮤니티(Builder 월 3만원 / Pro 월 8만원)를 런칭한다. 콘텐츠 → 뉴스레터 → 무료 커뮤니티 → 유료 커뮤니티 → 코칭/디지털 상품의 깔때기를 점진적으로 구축하며, AI 에이전트 자동화 스킬로 콘텐츠 제작과 커뮤니티 운영을 주 18시간 내에 가능하게 시스템화한다. 12개월 후 월 사이드 수입 300~600만원, 36개월 후 월 1,600~3,200만원을 목표로 하며, 이 시점에 자가 마련 자금 확보 + 정규직 졸업을 판단한다.

## 3. Phase별 목표와 행동

### Phase 1: 콘텐츠로 사람 모으기 (Month 1~6)

| 핵심 행동 | 구체적 내용 | 완료 기준 |
|-----------|------------|-----------|
| **1. 유튜브 쇼츠 일관 업로드** | 주 4개, 총 100개+. "직장인 AI 사이드 프로젝트" 주제. 60초 이내, 한국어. | 6개월간 100개 업로드 |
| **2. AI 자동화 파이프라인 구축** | 트렌드 수집 → 스크립트 생성 → 업로드 자동화. Python + Claude API + YouTube API. | 콘텐츠 제작 시간 50% 단축 |
| **3. 전환 퍼널 세팅** | 뉴스레터 (Beehiiv/Substack) 런칭. 쇼츠 → 뉴스레터 CTA. 카카오 오픈채팅 + Discord 무료 커뮤니티 개설. | 뉴스레터 구독자 300명+ |

### Phase 2: 커뮤니티 시작 → 유료 전환 (Month 7~18)

| 핵심 행동 | 구체적 내용 | 완료 기준 |
|-----------|------------|-----------|
| **1. 유료 커뮤니티 런칭** | Discord 유료 채널 (Builder 3만원/Pro 8만원). 주간 Q&A 라이브. 책임감 그룹 + 버디 시스템. | 유료 멤버 80명 |
| **2. 수익 다각화** | 디지털 상품 2개 출시. 그룹 코칭 격주 1회. 어필리에이트 셋업. | 월 사이드 수입 300만원+ |
| **3. 콘텐츠 시스템화** | 쇼츠 주 5개 (라이브 클립 포함). AI 파이프라인 고도화. 콘텐츠 버퍼 2주분 유지. | 콘텐츠 제작 시간 주 8시간 이내 |

### Phase 3: 커뮤니티 스케일 + 자가 마련 (Month 19~36)

| 핵심 행동 | 구체적 내용 | 완료 기준 |
|-----------|------------|-----------|
| **1. 커뮤니티 스케일** | 모더레이터 도입. Skool 또는 Insty 자체 플랫폼 이전. 유료 멤버 200~500명. | 커뮤니티 월 매출 1,000만원+ |
| **2. 자산 축적** | 월 사이드 수입 1,000만원+ → 저축 + 투자 → 자가 마련 자금 확보 | 자가 마련 자금 1억+ 축적 |
| **3. 정규직 졸업 준비** | 사이드 수입이 월급의 3배+ 유지 6개월 → 졸업 결정 | 월 사이드 수입 안정적 1,500만원+ |

### Phase 4: 풀타임 전환 (Month 37~96)

| 핵심 행동 | 구체적 내용 | 완료 기준 |
|-----------|------------|-----------|
| **1. 풀타임 전환** | 정규직 퇴사. 주 40시간 풀 투입. | 퇴사 완료 |
| **2. 팀 빌딩** | 파트타임/프리랜서 1~2명. 에디터, 커뮤니티 매니저. | 팀 운영 |
| **3. Insty 플랫폼 확장** | 자체 플랫폼으로 커뮤니티 + 콘텐츠 + 상품 통합. "디지털 건물주" 실현. | 플랫폼 MAU 5,000+ |

## 10. Sam에게 보내는 마지막 메시지

Sam, 솔직하게 말하겠다.

**이 계획은 쉽지 않다.** 퇴근하고 피곤한 몸으로 2시간 더 일하는 건 근성이 아니라 습관의 문제다. 처음 한 달은 의지로 버틸 수 있지만, 3개월째가 되면 "이걸 왜 하고 있지?"라는 생각이 반드시 든다. 그때 구독자가 200명이고 수입이 0원이면 더 그렇다.

**하지만 숫자를 봐라.**
- 보수적 시나리오에서도 1년 후 월 320만원, 3년 후 월 1,600만원.
- 지금 정규직만 하면 3년 후에도 월 445만원이다. 10년 뒤에도 크게 다르지 않다.
- 사이드를 시작하면 3년 후 월 2,000만원+가 가능하다.

**너의 진짜 무기는 AI 자동화 스킬이 아니다.** SLAM 엔지니어 + AI 에이전트 + 실제 빌딩 과정이라는 조합이 유일한 건 맞지만, 진짜 무기는 **"보여줄 수 있는 과정"** 자체다. 성공한 사람이 알려주는 건 넘쳐난다. 지금 만들고 있는 사람이 보여주는 건 드물다. 그게 차별화다.

**SaaS 만들지 마라. 도구 만들지 마라.** AI가 6개월 후에 다 만들어준다. "사람이 모이는 장소"만이 AI가 대체할 수 없다. 커뮤니티를 만들어라. 콘텐츠로 사람을 모으고, 신뢰로 붙잡아라.

**첫 달은 결과가 아니라 시스템을 만드는 달이다.** 구독자 0명이어도 성공이다 — 12개 쇼츠가 예약 업로드되어 있고, 뉴스레터가 세팅되어 있고, 자동화 파이프라인이 돌아가고 있으면. 그 시스템이 6개월 뒤에 복리로 돌아온다.

**자가 마련 + 결혼은 3년 안에 가능하다.** 보수적으로 1.5억, 낙관적으로 1.7억 축적 가능. 하지만 그건 3년간 멈추지 않았을 때의 이야기다.

오늘 퇴근하고 뭘 하나?
1. 유튜브 채널 만들어라.
2. 트위터 계정 만들어라.
3. 첫 쇼츠 스크립트를 써라.

그게 전부다. 나머지는 내일 하면 된다.

**시작이 반이 아니다. 시작이 전부다.** 시작하지 않으면 0이다. 시작하면 최소한 배운다. 그리고 그 배움이 다음 시도의 자산이 된다.

3년 후의 Sam이 지금의 Sam에게 감사할 수 있도록, 오늘 저녁 8시에 노트북을 열어라.
