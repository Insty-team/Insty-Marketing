## 3. Phase 1~2: AI 대행 서비스 (Month 1~12)

> **목표**: 현금 흐름 확보 + 업종별 AI 운영 패턴 데이터 축적

### 3.1 가격 철학 — 사장님이 "해볼까?"라고 느끼는 가격

소상공인의 현실:
- 월 매출 300~500만원, 순이익 150~250만원
- 네이버 예약, 카카오 채널 챗봇 등 기본 도구는 **이미 무료**
- 200만원 세팅비 = 순이익 1개월치 → **절대 안 냄**
- 월 20만원 구독 = "매달 20만원이나?" → **부담**

**돈을 낼 가치가 있는 것**: 무료 도구로는 **안 되는 것**

| 무료로 되는 것 (가치 없음) | 돈 낼 가치가 있는 것 |
|--------------------------|-------------------|
| 예약 받기 (네이버 예약) | 노쇼 고객 자동 전화/문자 + 블랙리스트 관리 |
| 인스타 포스팅 (Meta Suite) | 우리 가게 사진으로 매주 콘텐츠 자동 생성 + 해시태그 최적화 |
| 카톡 응대 (카카오 채널) | 새벽 문의에 자동 상담 + 예약까지 완료 |
| 매출 기록 (엑셀) | "젤네일 비중 높으니 재료 미리 주문하세요" 예측 |

### 3.2 수익 모델 — 저가 다수, 성과 기반

**하나의 구독 라인**: Phase 1에서 시작한 구독이 Phase 3~4 SaaS로 자연 전환됨.
고객 입장에서는 가격도 서비스도 안 끊김 — 뒤에서 수동→자동으로 바뀔 뿐.

| 수익원 | 단가 | 프레이밍 | 설명 |
|--------|------|---------|------|
| **초기 세팅** | **30~50만원/건** | "알바 한 달 비용" | Samwoo가 직접 세팅 (Phase 1~2) → 셀프 온보딩으로 전환 (Phase 3~4) |
| **월 구독** | **2.9~14.9만원/월** | "하루 천 원~" | Phase 1: Samwoo가 수동 관리 → Phase 3: SaaS 자동 관리 (가격 동일) |
| **성과 보수** | **노쇼 감소/매출 증가분의 10~20%** | "효과 없으면 안 내도 됨" | 리스크 제로 → 진입 허들 최소화 |
| **무료 템플릿** | 0원 (미끼) | "그냥 드릴게요" | 연락처 수집 → 육성 → 세팅 제안 |

**구독 플랜** (Phase 1부터 동일 구조):

| 플랜 | 월 구독료 | 프레이밍 | 포함 기능 |
|------|----------|---------|----------|
| **Basic** | **2.9만원/월** | "하루 천 원" | 예약관리 + 알림 자동화 |
| **Standard** | **7.9만원/월** | "알바 하루 시급" | + 마케팅 + 고객 분석 + CS |
| **Premium** | **14.9만원/월** | "알바 이틀 치" | + 멀티매장 + 커스텀 + 전담 지원 |

```
Phase 1~2: Samwoo가 세팅 + 수동 관리 (구독 플랜 동일)
Phase 3~4: SaaS로 전환 (고객은 변화를 못 느낌, Samwoo만 효율 UP)
```

```
                                         + SaaS 전환 자연스러움
```

### 3.5 세일즈 채널 전략

**소상공인은 "AI OS"를 검색하지 않는다.** 그들이 검색하는 것은 "노쇼 줄이는 법", "인스타 홍보하는 법" 같은 날것의 문제다. 크몽/숨고는 이미 해결책을 아는 사람이 모이는 곳이라 단가 경쟁에 빠진다.

**전략: 고객이 이미 모여있는 곳에서, 그들의 언어로, 시각적 결과물을 보여준다.**

| 채널 | 전략 | 기대 효과 |
|------|------|----------|
| **숏폼 영상** (YouTube/Instagram) | Before/After로 공감 → "나도 저거 필요해" | 폭발적 인지 + 인바운드 유입 |
| **커뮤니티 침투** (네이버 카페/카카오 OC) | 정보성 글로 신뢰 → "저요!" 댓글 → DM | 가장 높은 전환율 |
| **무료 미끼 상품** (PDF/템플릿) | 작은 문제 해결 → 연락처 수집 → 전체 자동화 제안 | 리드 수집 + 신뢰 구축 |

**금지 용어**: AI OS, 자동화 시스템, SaaS, 에이전트, 솔루션, 플랫폼
**필수 프레이밍**: "24시간 매니저", "알아서 해주는", "사장님 대신", "하루 3천원"

## 4. Phase 3~4: 셀프서비스 전환 (Month 10~24)

> **목표**: Samwoo 수동 관리 → SaaS 자동 관리로 전환. 고객 경험은 동일, 운영 효율만 UP.

### 4.1 전환 컨셉

Phase 1~2에서 반복된 세팅 패턴을 코드화하여 **셀프 온보딩 + 자동 관리**로 전환.

```
Phase 1~2                          Phase 3~4
Samwoo가 직접 세팅 (30~50만원) →   셀프 온보딩 (19.9만원) + 가이드
Samwoo가 수동 관리 (월 구독)    →   SaaS가 자동 관리 (동일 월 구독)
```

고객 입장에서 달라지는 것:
- 온보딩비가 30~50만 → **19.9만원**으로 내려감 ("20만원도 안 되는")
- 14일 무료 체험 추가 (카드 등록 불필요)
- 구독 플랜/가격은 **Phase 1과 동일** — 전환 마찰 제로

**"사장님 전용 24시간 매니저"** — 카톡에 말하면 알아서 해주는 AI

```
사장님: "오늘 예약 현황 알려줘"
AI: 오늘 예약 12건, 노쇼 위험 2건 (리마인드 발송 완료), 빈 슬롯 3개

사장님: "이번 주 매출 어때?"
AI: 이번 주 매출 320만원 (전주 대비 +12%), 인기 시술: 젤네일(45%)

사장님: "다음 주 인스타 포스트 만들어줘"
AI: 네, 시즌 트렌드 반영한 초안 3개 준비했습니다 [초안 링크]
```

> **주의**: 사장님에게 "AI OS", "COO SaaS"라고 말하면 안 됨.
> 내부적으로만 이렇게 부르고, 고객에게는 "24시간 매니저"로 포지셔닝.

### 4.2 매출 시뮬레이션

```
Month 10~12: SaaS 출시 + 기존 대행 고객 80명 중 50명 전환
             SaaS (50명 × 7.9만 avg) + 신규 온보딩 10건 (200만) + 대행 병행
             = 월 800~1,000만원

Month 13~18: SaaS 200명 + 신규 온보딩 15건/월
             SaaS (200명 × 7만 avg) + 온보딩 (15 × 20만) + 성과보수
             = 월 1,400~1,800만원 ← 목표 안정 돌파

Month 19~24: SaaS 400명 + Premium 비중 증가
             SaaS (400명 × 8만 avg) + 온보딩 + 성과보수
             = 월 2,000~2,500만원
```

**저가 모델의 핵심**: 고객 수가 빠르게 늘어남
- 고가 모델: M24에 150명 → MRR 2,250만 (높은 이탈 리스크)
- 저가 모델: M24에 400명 → MRR 3,200만 (낮은 이탈 + 입소문 + 네트워크 기반)

## 5. Phase 5: AI 에이전트 협업 네트워크 (Month 20~30)

> **목표**: 1인 기업가들의 AI 에이전트끼리 협업하는 네트워크 구축

### 5.1 컨셉

```
네일샵 AI: "이번 주 시술 후 꽃 선물 이벤트 하고 싶은데"
     ↕ (에이전트 네트워크)
꽃집 AI: "네일샵 고객 대상 미니 부케 패키지 제안드립니다 — 건당 2만원"
     ↕
네일샵 AI: "좋아요, 이번 주 예약 고객 30명에게 옵션 제안할게요"
```

1인 기업가는 별도 영업 없이, AI 에이전트가 자동으로 협업 기회를 찾고 제안.

### 5.2 수익 모델

| 수익원 | 모델 | 예상 수익 |
|--------|------|----------|
| **Premium 플랜 업그레이드** | 네트워크 기능은 Premium 전용 | 업그레이드 유도 |
| **거래 수수료** | 에이전트간 거래의 3~5% | 거래 규모에 비례 |
| **SaaS 구독 지속** | 기존 고객 유지 + 신규 | 3,200만원+ MRR |

### 5.3 매출 시뮬레이션

```
Month 20~24: 네트워크 베타 + SaaS 지속 성장
             SaaS 500명 + 거래 수수료 시작
             = 월 2,500~3,500만원

Month 25~30: 네트워크 100명+ 활성 + 거래 수수료 본격화
             = 월 3,500~5,000만원
```

### 5.4 필요 아키텍처 확장

| 새 에이전트 | 역할 |
|------------|------|
| **Network Agent** | 에이전트간 협업 매칭, 거래 중개, 품질 관리 |
| **Analytics Agent** | 플랫폼 전체 데이터 분석, 인사이트 보고 |

```
기존 10개 + Network Agent + Analytics Agent = 총 12개
```

## 6. Phase 6~7: 비즈니스 자동 런칭 + 자기발견 (Month 28~36+)

> **목표**: "아이디어만 있으면 AI가 사업을 런칭해주는" 플랫폼

### 6.1 컨셉

```
사용자: "동네에서 건강 도시락 배달 사업 하고 싶어"
플랫폼:
  1. 시장 조사 AI → 동네 수요 분석, 경쟁자 파악, 가격대 제안
  2. 브랜딩 AI → 이름, 로고, 메뉴 구성 제안
  3. 운영 AI → 주문 시스템, 배달 루트, 재료 발주 자동화
  4. 마케팅 AI → 인스타/당근마켓 홍보 자동 실행
  5. 네트워크 AI → 근처 식자재 공급업체 에이전트와 자동 계약
```

### 6.2 수익 모델

| 수익원 | 모델 | 비고 |
|--------|------|------|
| 런칭 패키지 | 100~500만원 (일회) | 복잡도에 따라 |
| 운영 구독 | 14.9~49.9만원/월 | 에이전트 수/기능에 따라 |
| 성공 수수료 | 매출의 1~3% | 선택적 |
| 네트워크 효과 | 누적 | 기존 Phase 5 네트워크와 연결 |

### 6.3 "자기 발견" 요소

단순히 사업 런칭만이 아니라, **"나에게 맞는 사업"을 찾아주는** 가치:

```
사용자의 경험/관심사/성향 분석
  → AI가 적합한 사업 모델 3~5개 제안
  → 각 모델의 시뮬레이션 (예상 수익, 필요 시간, 리스크)
  → 선택 후 자동 런칭
```
