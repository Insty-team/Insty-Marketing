# AI 에이전트 팀 관련 콘텐츠 전략

> 핵심 메시지: "혼자서도 충분히 만들 수 있습니다. 우리가 더 빠르게 해드릴 뿐."
> 목적: 교육 콘텐츠로 신뢰 구축 → "직접 할 시간 없는 분"이 서비스 구매

## 콘텐츠 퍼널

```
무료 교육 콘텐츠 (인식)
  → "나도 할 수 있겠다" (관심)
    → 직접 시도 → 시간이 너무 걸림 (고민)
      → "전문가가 해주면 3일 만에?" (구매)
```

이 퍼널이 작동하는 이유:
- 지식을 숨기지 않으므로 **신뢰** 생김
- 직접 시도해본 사람이 **가치를 이해**하고 구매
- 교육 자체가 **사회적 증명** (이 사람이 전문가)

## 시리즈 1: "AI로 1인 기업 팀 만들기" (한국어)

### 에피소드 구성

| # | 제목 | 핵심 내용 | CTA |
|---|------|---------|-----|
| 1 | 왜 1인 사업자에게 AI 팀이 필요한가 | 혼자 하는 사업의 한계, AI 팀의 개념 | 시리즈 팔로우 |
| 2 | 5분 만에 마케팅 에이전트 만들기 | 프롬프트 복붙으로 바로 사용, 실제 결과 공유 | 프롬프트 템플릿 링크 |
| 3 | 영업 에이전트: DM을 대신 써주는 AI | DM 템플릿, 개인화 방법, 실제 예시 | 무료 진단 신청 |
| 4 | CEO 에이전트: 업무 분배를 AI에게 | 오케스트레이터 개념, 사용법, 실제 시연 | 풀 시스템 소개 |
| 5 | 하루 30분으로 돌리는 AI 비즈니스 루틴 | 아침 루틴 워크플로우, 실제 하루 시연 | 서비스 안내 |

### 채널별 변환


## 시리즈 3: "Build in Public — AI Team" (English)

### Week 1~4 스토리라인

| 주 | 주제 | 콘텐츠 |
|---|------|--------|
| 1 | "I built an AI team" | 에이전트 팀 구축 과정 공유, 첫 결과 |
| 2 | "Week 1 results" | 실제 수치 (리드 수, 콘텐츠 수, 시간 절약) |
| 3 | "Selling AI teams" | 서비스 판매 시작, 첫 고객 경험 |
| 4 | "Month 1 review" | 매출, 배운 점, 다음 계획 |

### 핵심 메시지 (영문 콘텐츠)
- "Solopreneurs don't need to hire. They need AI agents."
- "4 prompts = 4 team members. Cost: $0~20/month."
- "I'm building an AI-powered business in public. Follow along."

## 콘텐츠 → 서비스 연결 포인트

### 무료 제공 (리드 자석)
- 범용 에이전트 팀 프롬프트 (solopreneur-general)
- "5분 만에 마케팅 에이전트 만들기" 가이드

### 유료 전환 포인트
- "직접 만들어봤는데 시간이 너무 걸려요" → D1 (₩30~50만)
- "업종에 맞게 커스텀하고 싶어요" → D2 (₩60~100만)
- "전체 비즈니스 시스템을 세팅해주세요" → D3 (₩150~200만)

### 판매 메시지
```
"위 콘텐츠에서 본 AI 에이전트 팀, 직접 만드실 수 있습니다.
시간이 부족하시면 제가 3일 안에 만들어드립니다.
비즈니스 맞춤 + 워크플로우 + 사용 가이드 포함.
무료 15분 진단 → DM으로 '에이전트' 보내주세요."
```
