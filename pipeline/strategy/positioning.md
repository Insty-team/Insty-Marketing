# 포지셔닝 & 핵심 메시지

## 원라이너

**한국어**: "강의 말고, 시스템. 1주일 안에 세일즈 파이프라인을 직접 만들어드립니다."

**영어**: "Not a course. Not a template. A working sales pipeline — built for you in 7 days."

```
"1인 사업자분들이 실력은 좋은데

세일즈 강의 듣고 도구 세팅하고 하면 한 달이 걸리는데,

리드 발굴, 콘텐츠, 아웃리치, CRM까지

50만원부터이고, 30분 무료 진단도 해드려요."
```

## 핵심 차별점 3가지 (경쟁사 대비)

### 1. "도구가 아니라 완성된 시스템"
- SaaS → 도구만 줌 (세팅은 알아서)
- 강의/코칭 → 지식만 줌 (실행은 알아서)
- 크몽 프리랜서 → 단편 작업만 해줌
- **우리 → 전략 설계 + 구축 + 운영 가이드까지 원스톱**

### 2. "1주일 안에 완성"
- DIY → 2~4주 + 시행착오
- 에이전시 → 납기 불명확, 월 리테이너
- 코칭 → 3~6개월 과정
- **우리 → AI(Claude Code) 활용으로 7일 납기 보장**

### 3. "구축 후 자립"
- 에이전시 → 계속 의존 (월 비용 발생)
- SaaS 구독 → 매달 비용 누적
- **우리 → 1회 구축비, 이후 Notion + 무료 도구로 자체 운영**

## 타겟별 핵심 메시지

| 세그먼트 | 페인포인트 | 핵심 메시지 |
|---|---|---|
| 프리랜서 | 일감 들쭉날쭉, 수동 영업 | "다음 달 매출 걱정 없이, 자동으로 문의가 들어오는 시스템" |
| 코치/컨설턴트 | 고객 유치 방법 모름 | "전문성은 있는데 세일즈가 어려운 분을 위한 맞춤 파이프라인" |
| 크리에이터 | 팔로워 ≠ 매출 | "팔로워를 고객으로 바꾸는 전환 시스템" |
| 온라인 강사 | 플랫폼 수수료 70% | "수강생을 직접 고객으로 만드는 자체 세일즈 채널" |
| 스몰비즈 | 마케팅에 시간 없음 | "하루 30분으로 세일즈가 돌아가는 자동화 시스템" |

## 블루오션 근거

경쟁사 분석 결과, **한국 시장에 아래 조합을 제공하는 플레이어가 없음**:

1. 1인사업자 전용 (에이전시는 기업 대상, 크몽은 단발성)
2. 세일즈 파이프라인 "풀셋" 구축 (부분이 아닌 전체)
3. AI 기반 빠른 납기 (1주일)
4. 1회 고정가 (월 리테이너 X)
5. 구축 후 자립 가능 (의존 X)

> 프프랩이 가장 유사하지만 오프라인 업종(병의원, F&B) 중심.
> 온라인 기반 솔로프리너 대상은 사실상 블루오션.
