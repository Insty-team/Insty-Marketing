# Phase 1 실행 계획: "파이프라인 대행 서비스"

> 목표: 4주 안에 첫 매출 발생, 서버비 충당
> 시작일: 2026-03-10 (내일)

## 전략 방향 (확정)

| 영역 | 타겟 | 언어 |
|---|---|---|
| 서비스 대행 (Phase 1) | **한국** 솔로프리너 | 한국어 |
| 콘텐츠/빌드인퍼블릭 | **북미** 오디언스 빌딩 | 영어 |

## 1. 오퍼 (정확히 뭘 파는가)

### 서비스명: "AI 세일즈 파이프라인 구축 서비스"

> "당신의 비즈니스에 맞는 AI 자동화 세일즈 파이프라인을 1주일 안에 만들어드립니다"

### 딜리버리 (클라이언트가 받는 것)
1. **리드 발굴 자동화** — 타겟 고객 리스트 + 자동 수집 시스템
2. **콘텐츠 자동화** — 인스타/링크드인/X 포스트 자동 생성 파이프라인
3. **아웃리치 템플릿** — DM/이메일 시퀀스 (개인화 자동화)
4. **CRM 세팅** — Notion DB 기반 고객 관리 시스템
5. **주간 리포트** — 자동 성과 추적

### 왜 이 오퍼인가
- Claude Code로 실제로 빠르게 만들 수 있는 것들
- 솔로프리너가 가장 못하는 것 = 체계적 세일즈
- 결과물이 명확하고 측정 가능
- 1주 안에 딜리버리 가능 (Claude Code 활용)

---

## 2. ICP (누구한테 파는가)

### 1차 타겟: 이미 스킬은 있지만 세일즈 시스템이 없는 사람

| 세그먼트 | 예시 | 페인포인트 |
|---|---|---|
| 프리랜서 | 개발자, 디자이너, 마케터 | 일감이 들쭉날쭉, 수동 영업 |
| 코치/컨설턴트 | 비즈니스, 커리어 코치 | 고객 유치 방법 모름 |
| 크리에이터 | 유튜버, 블로거 | 팔로워는 있는데 수익화 못함 |
| 스몰비즈 | 온라인 쇼핑몰, 1인 에이전시 | 마케팅에 시간 못 씀 |

### 2차 필터 (구매 가능성 높은 사람)
- 이미 수익이 있음 (월 100만원+)
- SNS 활동 중 (채널이 있음)
- "시스템"이 없다는 걸 자각하고 있음
- 한국 거주 + 한국어 소통 가능

## 4. 에이전트 팀 구조

> Claude Code 기반으로 4개 팀을 운영. 각 팀은 프롬프트 + 워크플로우로 구성.

### Team Research (리서치팀)
- **역할**: ICP 리서치, 경쟁사 분석, 채널 분석
- **도구**: WebSearch, WebFetch
- **산출물**: ICP 보고서, 경쟁 분석서, 채널 추천서
- **언제 쓰나**: 새 클라이언트 온보딩 시, 주간 시장 동향 파악

### Team Sales (세일즈팀)
- **역할**: 잠재 고객 발굴, DM/이메일 작성, 프로포절 생성
- **도구**: WebSearch (고객 리서치), 프롬프트 (메시지 생성)
- **산출물**: 타겟 리스트, 아웃리치 메시지, 프로포절 문서
- **언제 쓰나**: 매일 — 아웃리치 루틴

### Team Content (콘텐츠팀)
- **역할**: 빌드인퍼블릭 콘텐츠 생성, SNS 포스트 작성
- **도구**: 프롬프트 템플릿
- **산출물**: 일간 포스트 (X/링크드인), 주간 긴 글, 사례 스터디
- **언제 쓰나**: 매일 1포스트 + 주 1 긴 글

### Team Delivery (딜리버리팀)
- **역할**: 클라이언트 파이프라인 실제 구축
- **도구**: Claude Code (코드 생성), Notion API, 자동화 툴
- **산출물**: 완성된 파이프라인 (리드→콘텐츠→아웃리치→CRM)
- **언제 쓰나**: 계약 체결 후
