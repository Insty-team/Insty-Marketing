# Team Sales 플레이북

> 목표: 한국 솔로프리너 대상 잠재 고객 발굴 → 미팅 → 클로징
> 타겟 시장: 한국 (한국어 소통)

---

## 일일 루틴

### 1. 타겟 리서치 (30분)
```
프롬프트: "아래 조건에 맞는 한국 솔로프리너/프리랜서를 X(트위터)에서 찾아줘:
- 이미 수익이 있는 사람 (서비스/제품 판매 중)
- 최근 7일 내 활동
- '고객 유치', '마케팅', '세일즈', '자동화' 관련 고민을 언급한 사람
- 팔로워 500~10,000명 (너무 작지도 크지도 않은)
각 사람에 대해: 이름, 프로필 링크, 하는 일, 발견한 페인포인트 정리"
```

### 2. 아웃리치 메시지 생성 (20분)

#### DM 템플릿 A: 가치 제공형
```
안녕하세요 {이름}님!

{그 사람의 최근 활동/콘텐츠 언급} 잘 봤습니다.

저도 1인 사업 하면서 세일즈 파이프라인을 AI로 자동화하는 걸 하고 있는데요,
{이름}님 비즈니스에 맞는 간단한 진단을 무료로 해드릴 수 있을 것 같아서 연락드립니다.

- 어디서 잠재 고객을 찾을 수 있는지
- 자동화할 수 있는 반복 작업이 뭔지
- 10분이면 됩니다

관심 있으시면 편하게 답 주세요!
```

#### DM 템플릿 B: 사례 공유형
```
안녕하세요 {이름}님!

저는 솔로프리너분들 AI 세일즈 파이프라인 만들어드리는 일을 하고 있어요.
최근에 {비슷한 업종} 하시는 분 파이프라인을 만들어드렸는데,
{구체적 결과: 예) 아웃리치 시간 80% 절감} 되더라고요.

{이름}님 {하시는 일}에도 비슷하게 적용 가능할 것 같은데,
혹시 10분 정도 이야기 나눠볼 수 있을까요?
```

#### DM 템플릿 C: 커뮤니티 참여형
```
{커뮤니티/스레드에서의 맥락 언급}

저도 비슷한 고민을 했었는데, AI로 {구체적 해결 방법}을 만들어서 해결했어요.
관심 있으시면 제가 어떻게 했는지 공유해드릴게요.
```

### 3. 팔로업 (10분)
- 1차 DM 후 48시간 무응답 → 가벼운 팔로업
- 관심 표현 → 미팅 일정 잡기
- 거절 → 감사 인사 + 콘텐츠 링크 공유

---

## 미팅 스크립트 (15분 구조)

### 0~3분: 라포
- 상대방 비즈니스에 대한 진심 어린 관심

### 3~8분: 진단 (질문)
1. "지금 새로운 고객은 주로 어떻게 오시나요?"
2. "한 달에 몇 명 정도 새 고객이 오시나요?"
3. "고객 유치에 하루 얼마나 시간 쓰시나요?"
4. "가장 힘든 부분이 뭔가요?"
5. "이게 자동화되면 어떨 것 같으세요?"

### 8~12분: 솔루션 제안
- 진단 결과 기반으로 맞춤 파이프라인 설명
- 스크린 공유로 비슷한 사례 보여주기
- "이런 걸 1주일 안에 만들어드릴 수 있습니다"

### 12~15분: 클로징
- 가격 안내 (스타터/그로스/프리미엄)
- "이번 주 중으로 시작하시면 론칭 특가로 해드릴게요"
- 결정 데드라인 제공 (3일)

## 프로포절 템플릿

```markdown
# {클라이언트명} AI 세일즈 파이프라인 제안서

## 현재 상황 진단
- {진단 미팅에서 파악한 문제점 3가지}

## 제안 솔루션
| 구성 요소 | 내용 | 기대 효과 |
|---|---|---|
| 리드 발굴 | {맞춤 채널 + 자동 수집} | 월 {X}명 잠재 고객 확보 |
| 콘텐츠 자동화 | {SNS 포스트 자동 생성} | 일 {X}분으로 콘텐츠 발행 |
| 아웃리치 | {개인화 DM/이메일 시퀀스} | 응답률 {X}% 목표 |
| CRM | {Notion 기반 고객 관리} | 파이프라인 가시화 |

## 타임라인
- Day 1-2: 리서치 + 전략 설계
- Day 3-5: 파이프라인 구축
- Day 6-7: 테스트 + 인수인계

## 투자 비용
- {선택 패키지}: ₩{금액}
- 포함: {리스트}
- 월 유지보수 (선택): ₩30만/월

## 보장
- 7일 내 완성 보장
- 만족하지 않으면 50% 환불
```
