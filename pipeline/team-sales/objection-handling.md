# 반론 처리 가이드

> 세일즈 미팅/DM에서 자주 나오는 반론과 대응 스크립트

---

## 가격 관련

### "너무 비싸요"
```
"이해합니다. 혼자 만드실 수도 있어요.
다만 도구 리서치 + 세팅 + 테스트에 보통 2~4주 걸리고,
그 시간에 본업으로 벌 수 있는 돈을 생각하시면
50만원은 빠르게 회수되실 거예요.

그리고 지금 론칭 초기라 이 가격인 거고, 곧 올릴 예정이에요."
```

### "효과가 있을지 모르겠어요"
```
"당연한 걱정이에요. 그래서 무료 진단을 먼저 드리는 거예요.
진단 결과 보시고 '이거 되겠다' 싶으시면 진행하시면 됩니다.

만약 파이프라인 만들어드리고 만족 못하시면 50% 환불해드리고요."
```

### "나중에 할게요"
```
"물론이요! 다만 하나만 여쭤볼게요 —
지금 한 달에 고객 유치에 몇 시간 쓰세요?

그 시간이 한 달 더 계속된다는 건데,
1주일 투자로 그걸 절반으로 줄일 수 있다면 어떨까요?"
```

---

## 신뢰 관련

### "어떻게 1주일 만에 가능해요?"
```
"Claude Code라는 AI 도구로 구축하기 때문이에요.
보통 에이전시가 2~4주 걸리는 걸 AI가 코드/자동화를 빠르게 만들어줘서
저는 전략과 커스터마이징에 집중할 수 있어요.

실제로 제 비즈니스 파이프라인도 이렇게 만들어서 운영 중이에요."
```

### "사례가 있나요?"
```
[초기에 사례가 없을 때]
"솔직히 이 서비스는 막 시작해서 외부 사례는 아직 없어요.
대신 제 자체 비즈니스에 적용한 결과를 보여드릴 수 있고,
그래서 지금 론칭 특가로 드리는 거예요.

첫 고객분이시면 특별히 더 신경 써서 만들어드릴게요."

[사례가 생긴 후]
"네! {업종} 하시는 분 사례가 있는데요,
파이프라인 구축 후 {구체적 결과}가 나왔어요.
사례 자료 보내드릴까요?"
```

### "AI로 만든 게 진짜 돌아가나요?"
```
"좋은 질문이에요. AI가 만드는 건 틀이고,
거기에 {이름}님 비즈니스에 맞게 제가 직접 커스터마이징해요.

리드 발굴 방법, 메시지 톤, CRM 구조 — 이런 건 사람이 판단해야 하거든요.
AI는 속도, 전략은 저, 비즈니스 이해는 {이름}님. 이 조합이에요."
```

## 서비스 이해 관련

### "정확히 뭘 해주는 건가요?"
```
"4가지를 만들어드려요:

1. 리드 발굴 — {이름}님 고객이 될 사람을 자동으로 찾는 시스템
2. 콘텐츠 — SNS 포스트를 빠르게 만들 수 있는 자동화
3. 아웃리치 — 잠재 고객에게 보낼 DM/이메일 시퀀스
4. CRM — 누구한테 연락했고, 다음에 뭘 해야 하는지 한눈에 보는 대시보드

이 4개가 하나의 파이프라인으로 연결돼서,
매일 30분만 투자하면 세일즈가 돌아가게 해드리는 거예요."
```

### "나는 IT를 잘 모르는데요"
```
"전혀 문제없어요! Notion이랑 SNS만 쓸 줄 아시면 됩니다.
복잡한 건 다 저희가 만들어놓고,
{이름}님은 매일 30분 루틴만 따라하시면 돼요.

인수인계 때 1시간 동안 하나하나 다 알려드리고,
가이드 문서도 드려요."
```

## 전환 멘트 (클로징)

### 부드러운 클로징
```
"오늘 말씀 나눠보니 {핵심 문제}가 가장 크시네요.
이거 해결하는 게 먼저일 것 같은데,
이번 주 중으로 시작하시면 다음 주에 파이프라인 쓰실 수 있어요.
어떠세요?"
```

### 긴급성 부여
```
"지금 동시에 3명까지만 받고 있어서, 이번 주 자리가 {N}개 남았어요.
관심 있으시면 오늘 내일 중으로 말씀해주시면 바로 시작해드릴게요."
```
