# 무료 파이프라인 진단 프레임워크

> 15분 무료 진단 → 유료 전환의 핵심 도구
> DM에서 "무료 진단 해드릴게요"로 미팅 유도 시 사용

---

## 진단 전 준비 (5분)

클라이언트 SNS/웹사이트를 미리 살펴보고 아래 항목 체크:

```
프롬프트: "{클라이언트 이름/SNS 링크}의 비즈니스를 사전 분석해줘.

확인할 것:
1. 뭘 파는지 (제품/서비스)
2. 타겟 고객이 누구인지
3. 현재 사용 중인 채널 (SNS, 웹사이트, 블로그 등)
4. 콘텐츠 발행 빈도
5. 눈에 보이는 세일즈 퍼널이 있는지 (CTA, 링크, 문의 방법)
6. 개선 가능한 포인트 3가지 (구체적으로)"
```

---

## 진단 미팅 질문 (15분)

### Part 1: 현재 상황 파악 (5분)

| # | 질문 | 메모할 것 |
|---|---|---|
| 1 | "지금 하시는 일을 한 줄로 소개해주세요" | 서비스/제품 정의 |
| 2 | "새 고객은 주로 어떻게 오시나요?" | 유입 채널 |
| 3 | "한 달에 신규 고객이 몇 명 정도?" | 현재 볼륨 |
| 4 | "고객 유치에 하루/주 얼마나 시간 쓰세요?" | 시간 투자 |
| 5 | "지금 쓰고 계신 도구가 있나요?" | 기존 스택 |

### Part 2: 페인포인트 (5분)

| # | 질문 | 메모할 것 |
|---|---|---|
| 6 | "세일즈/마케팅에서 가장 힘든 부분은?" | 핵심 페인 |
| 7 | "과거에 시도해봤는데 안 된 것이 있나요?" | 실패 경험 |
| 8 | "이상적으로는 어떤 상태가 되고 싶으세요?" | 목표 상태 |

### Part 3: 진단 결과 공유 (5분)

```
"말씀 들어보니 3가지가 보여요:

1. [구체적 문제점 1] — 예: 리드 발굴이 수동이라 시간이 많이 드시네요
2. [구체적 문제점 2] — 예: 콘텐츠가 불규칙해서 인바운드가 약하시고
3. [구체적 문제점 3] — 예: CRM이 없어서 팔로업이 빠지시네요

이걸 해결하려면 [리드 자동화 + 콘텐츠 루틴 + CRM]이 필요한데,
저희가 1주일 안에 이걸 다 만들어드릴 수 있어요.

관심 있으시면 구체적인 제안서 보내드릴까요?"
```

---

## 진단 후 프로세스

### 관심 있음 → 24시간 내 프로포절 발송
```
프롬프트: "아래 진단 결과를 바탕으로 맞춤 프로포절을 작성해줘.
(team-sales/playbook.md의 프로포절 템플릿 사용)

클라이언트 정보:
- 이름: {이름}
- 업종: {업종}
- 진단 결과:
  1. {문제점 1}
  2. {문제점 2}
  3. {문제점 3}
- 목표: {원하는 상태}
- 기존 도구: {사용 중인 것}"
```

### 고민 중 → 3일 후 팔로업
```
"안녕하세요 {이름}님!
지난번 진단 때 말씀드린 {핵심 문제점}이 계속 생각나서요.
비슷한 상황이셨던 {업종} 분이 파이프라인 만드시고 {구체적 결과}가 나왔어요.
혹시 궁금하신 점 있으시면 편하게 말씀해주세요!"
```

### 거절 → 감사 인사 + 콘텐츠 연결
```
"아, 괜찮습니다! 혹시 나중에 관심 생기시면 언제든 연락주세요.
그리고 혼자 해보실 때 참고하실 수 있는 글 하나 공유드릴게요: {콘텐츠 링크}"
```
