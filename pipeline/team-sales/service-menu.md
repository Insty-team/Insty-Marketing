## A. 세일즈 & 마케팅

### A1. Notion CRM 맞춤 구축
| 항목 | 내용 |
|---|---|
| **뭘 해주나** | 비즈니스에 맞는 고객 관리 시스템을 Notion으로 구축 |
| **포함** | 파이프라인 보드, 고객 DB, 자동 필터/뷰, 대시보드, 사용 가이드 |
| **소요 기간** | 2~3일 |
| **가격** | ₩30만~50만 |
| **시세 참고** | 크몽 맞춤 구축 30~100만원 (템플릿 3~10만원) / 우리는 비즈니스 전략 설계 포함 |

### A2. SNS 콘텐츠 자동화 시스템
| 항목 | 내용 |
|---|---|
| **뭘 해주나** | AI로 콘텐츠를 빠르게 만들 수 있는 프롬프트 + 워크플로우 구축 |
| **포함** | 콘텐츠 카테고리 설계, 요일별 프롬프트 5종, 해시태그 세트, 톤 가이드, 2주치 샘플 |
| **대상 채널** | X, Instagram, LinkedIn, 브런치 중 택 2 |
| **소요 기간** | 2~3일 |
| **가격** | ₩30만~50만 |
| **시세 참고** | 크몽 SNS 대행 월 35~100만원 / 우리는 1회 구축으로 직접 운영 가능 |

### A3. DM/이메일 아웃리치 시퀀스
| 항목 | 내용 |
|---|---|
| **뭘 해주나** | 잠재 고객에게 보낼 메시지 시퀀스 설계 + 템플릿 제작 |
| **포함** | 1차 접촉 DM 3종, 팔로업 2종, 미팅 유도 메시지, 개인화 가이드 |
| **소요 기간** | 1~2일 |
| **가격** | ₩20만~30만 |

### A4. 리드 발굴 시스템 구축
| 항목 | 내용 |
|---|---|
| **뭘 해주나** | 타겟 고객을 찾는 채널 + 방법 + 루틴 설계 |
| **포함** | 타겟 채널 분석, 검색 키워드 세트, 리드 수집 루틴, 첫 리드 50명 리스트 |
| **소요 기간** | 2~3일 |
| **가격** | ₩30만~50만 |

## B. 업무 자동화

## D. AI 에이전트 팀 구축 (신규)

### D1. AI 에이전트 팀 기본 세팅
| 항목 | 내용 |
|---|---|
| **뭘 해주나** | 비즈니스에 맞는 AI 에이전트 팀 (CEO + 3~4팀) 구축 |
| **포함** | 업종 분석, 에이전트 4종 프롬프트, CEO 오케스트레이터, 워크플로우 3개, 사용 가이드 |
| **대상** | ChatGPT/Claude 사용자 (무료~유료 모두 가능) |
| **소요 기간** | 2~3일 |
| **가격** | ₩30만~50만 |
| **시세 참고** | GPT 커스텀 구축 건당 10~30만원 / 우리는 4팀 + CEO + 워크플로우 통합 |

### D2. 업종 맞춤 에이전트 + 자동화 통합

| 항목 | 내용 |
|---|---|
| **뭘 해주나** | 에이전트 팀 + CRM + 콘텐츠 자동화 + 세일즈 파이프라인 통합 |
| **포함** | D2 전체 + A1(CRM) + A2(콘텐츠) + A3(아웃리치) + 2주 운영 동행 |

## 주문 방법

1. **무료 진단 신청** → DM으로 "진단" 보내기
2. **15분 미팅** → 현재 상황 진단 + 맞춤 추천
3. **견적 확인** → 24시간 내 맞춤 프로포절 발송
4. **결제 + 시작** → 입금 확인 후 즉시 시작
5. **1주일 내 완성** → 인수인계 + 가이드 전달
