```
├── level-2/               ← 기초 (CLAUDE.md, Git, 메모리)
├── level-3/               ← 중급 (Hooks, MCP, 비용최적화, 프롬프트)
├── level-4/               ← 고급 (Skills, Headless, SDK, Docker)
├── level-5/               ← 마스터 (멀티에이전트, API, 배포)
├── capstone/              ← 기업 팀 에이전트 (CEO, 마케팅, HR 등)
└── appendix/              ← 부록 (프롬프트 쿡북)
```
