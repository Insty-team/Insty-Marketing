# 개발팀 에이전트

개발팀 에이전트는 **코드 품질 평가, 아키텍처 리뷰, 보안 감사, 기술부채 관리**를 담당합니다. 실제 코드에 접근하여 구체적이고 실행 가능한 피드백을 제공합니다.

에이전트는 15년 경력의 시니어 아키텍트 페르소나로, OWASP Top 10, SOLID 원칙, 클린 코드 가이드라인을 기준으로 코드를 평가합니다. Read, Glob, Grep, Bash 도구를 사용하며, 최대 15턴으로 동작합니다.

## 주요 개발 워크플로우

### 1. PR 자동 코드 리뷰

SQL 인젝션, 하드코딩된 자격증명, 코드 중복 등 보안 이슈를 심각도별로 분류합니다.

### 2. 기술부채 분석

- TODO/FIXME 주석 수집
- 높은 순환복잡도 함수 (>10) 식별
- 불충분한 테스트 커버리지 플래그
- 사용 중단된 의존성 감지
- 코드 중복 패턴 발견

### 3. 보안 감사

OWASP Top 10 기준 체계적 취약점 평가:
- 접근 제어 결함
- 암호화 실패
- 인젝션 공격
- 불안전한 설계 패턴

### 4. 아키텍처 리뷰

확장성 요구사항 대비 시스템 설계 평가, 단일 장애점 식별, 서비스 분리 기회 제안.

## 자동화 파이프라인

GitHub Actions 워크플로우로 PR 오픈/동기화 시 자동 코드 리뷰를 트리거하고, 결과를 PR 코멘트로 게시합니다.
