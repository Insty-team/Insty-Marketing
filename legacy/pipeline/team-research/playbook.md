# Team Research 플레이북

> 목표: ICP 리서치, 경쟁 분석, 시장 인사이트 제공

---

## 1. ICP 리서치 프롬프트

### 한국 솔로프리너 커뮤니티 맵핑
```
프롬프트: "한국에서 활동하는 솔로프리너/1인 사업자 커뮤니티를 리서치해줘.

찾아야 할 것:
1. X(트위터) — 한국어 솔로프리너 인플루언서/커뮤니티 (10명+)
2. 링크드인 — 1인 사업 관련 그룹/인플루언서
3. 오픈카톡방 — 프리랜서, AI, 사이드프로젝트 키워드
4. 디스코드/슬랙 — 한국 인디해커 커뮤니티
5. 커뮤니티 사이트 — 디스콰이엇, 인디해커스코리아 등

각 항목에 대해:
- 이름/링크
- 규모 (회원 수)
- 활성도 (최근 활동)
- 우리 타겟이 있을 확률 (높/중/낮)
- 진입 방법 (가입만? 초대 필요?)"
```

### 경쟁사/대체재 분석
```
프롬프트: "AI 세일즈 파이프라인 구축 서비스의 경쟁사/대체재를 분석해줘.

카테고리별로:
1. 직접 경쟁 — 비슷한 서비스를 하는 한국 사업자/에이전시
2. 간접 경쟁 — 세일즈 자동화 SaaS (한국/해외)
3. DIY 대체재 — 클라이언트가 직접 할 수 있는 방법

각 항목에 대해:
- 서비스명
- 가격대
- 타겟 고객
- 우리와의 차별점
- 우리가 이길 수 있는 포인트"
```

## 2. 클라이언트 온보딩 리서치

> 새 클라이언트 계약 시 Delivery 팀에 넘기기 전에 수행

```
프롬프트: "{클라이언트 업종/비즈니스} 분야에 대해 세일즈 파이프라인 구축을 위한 리서치를 해줘.

조사 항목:
1. 업종 특성
   - 일반적인 고객 획득 채널
   - 평균 고객 획득 비용 (CAC)
   - 세일즈 사이클 길이

2. 타겟 고객 (클라이언트의 고객)
   - 주요 페인포인트
   - 의사결정 과정
   - 자주 활동하는 온라인 채널

3. 콘텐츠 전략
   - 이 업종에서 효과적인 콘텐츠 유형
   - 키워드/해시태그 추천
   - 경쟁자들의 콘텐츠 전략

4. 자동화 기회
   - 반복되는 작업 목록
   - 자동화 가능한 도구 추천 (무료/저가 우선)"
```

## 3. 주간 시장 동향 리서치

매주 월요일 실행:
```
프롬프트: "이번 주 솔로프리너/1인사업 관련 트렌드를 리서치해줘.

1. X(트위터)에서 화제가 된 솔로프리너 관련 토픽
2. 새로 나온 AI 자동화 도구 (세일즈/마케팅 관련)
3. 한국 1인사업 시장 뉴스
4. 콘텐츠 아이디어로 쓸 수 있는 것 3가지"
```
