# Team Content 플레이북

> 목표: 영어 빌드인퍼블릭으로 북미 오디언스 빌딩 + 한국어 콘텐츠로 영업 보조

---

## 투트랙 전략

| 트랙 | 언어 | 채널 | 목적 |
|---|---|---|---|
| **빌드인퍼블릭** | 영어 | X, LinkedIn, Medium | 북미 오디언스, Insty 인지도 |
| **영업 보조** | 한국어 | 브런치, 오픈카톡 | 한국 고객 신뢰 구축 |

> 같은 경험을 영어/한국어 양쪽으로 가공. 작업은 1번, 콘텐츠는 2배.

---

## 포지셔닝
- **영어**: "Building AI-powered sales pipelines for solopreneurs — sharing the journey"
- **한국어**: "AI로 1인 사업자 세일즈 파이프라인을 만들어주는 사람"
- **차별점**: 실제 만들어주는 과정을 날것으로 공개
- **톤**: 실전적, 솔직, 과정 중심 (결과만 자랑 X)

---

## 일일 콘텐츠 루틴

### 영어 콘텐츠 — X / LinkedIn (매일 1포스트)

#### 콘텐츠 카테고리 (로테이션)

**Mon: Insight / Lesson**
```
Prompt: "Write a tweet about what I learned today from {work done}.
- Under 280 characters
- Include specific numbers/examples
- End with a question to spark engagement
- Hashtags: #buildinpublic #solopreneur
- Tone: honest, no fluff"
```

**Tue: Build in Public Update**
```
Prompt: "Write a build-in-public tweet about today's work:
- What I did (1 line)
- Why (1 line)
- Result/metric (1 line)
- What's next (1 line)
- Raw, authentic feel
- Hashtags: #buildinpublic"
```

**Wed: Tips / How-to Thread**
```
Prompt: "Write an X thread about {topic} for solopreneurs.
- Hook tweet (1 tweet)
- 3-5 key tips (1 tweet each)
- CTA tweet: 'I build AI sales pipelines for solopreneurs. DM me.'
- Each tweet under 280 chars"
```

**Thu: Case Study / Results**
```
Prompt: "Write a tweet sharing results from {client case or experiment}.
- Before: what the situation was
- After: what changed
- How: brief method
- Include specific numbers"
```

**Fri: Question / Engagement**
```
Prompt: "Write a question tweet that resonates with solopreneurs.
- About sales/marketing/automation struggles
- Make people want to reply
- Under 280 chars"
```

### 한국어 콘텐츠 — 영업 보조 (주 2~3회)

> 영어 콘텐츠를 한국 맥락으로 변환. 새로 쓰지 않음.

```
프롬프트: "아래 영어 포스트를 한국 1인 사업자 대상으로 한국어 버전으로 변환해줘.
- 한국 시장 맥락에 맞게 조정
- 브런치/오픈카톡 톤
- CTA: '무료 파이프라인 진단 해드립니다. DM 주세요'

원문: {영어 포스트}"
```

---

## 주간 장문 콘텐츠

### 영어 — LinkedIn / Medium (매주 금요일)

```
Prompt: "Write a weekly build-in-public recap post.

This week's work:
{weekly summary}

Format:
1. Key numbers (outreach Xpp, responses X, meetings X)
2. Biggest lesson
3. What failed + why
4. Next week's plan
5. Honest feelings (struggles, excitement)

Tone: vulnerable, honest. Show failures too.
Length: 500-800 words
CTA: 'I build AI sales pipelines for solopreneurs. DM me if you want one.'"
```

### 한국어 — 브런치 (격주)
> 영어 회고를 한국 맥락으로 변환 + 한국 클라이언트 사례 추가

---

## 사례 스터디 템플릿

```markdown
# {클라이언트 업종} 파이프라인 구축 사례

## 클라이언트 소개
- 업종: {업종}
- 규모: 1인
- 상황: {문제 상황}

## Before
- 고객 유치 방법: {수동/없음}
- 월 신규 고객: {X}명
- 마케팅 시간: 주 {X}시간

## 구축한 파이프라인
1. {리드 발굴}: {구체적 방법}
2. {콘텐츠}: {자동화 내용}
3. {아웃리치}: {시퀀스 설명}
4. {CRM}: {관리 방법}

## After (2주 후)
- 리드: 월 {X}명 자동 발굴
- 콘텐츠: 일 {X}분으로 발행
- 아웃리치 응답률: {X}%
- 신규 고객: {X}명

## 클라이언트 피드백
> "{실제 피드백}"
```

## 콘텐츠 캘린더 관리

| 필드 | 내용 |
|---|---|
| 날짜 | 발행일 |
| 채널 | X / 링크드인 / 브런치 |
| 카테고리 | 인사이트 / 빌드인퍼블릭 / 팁 / 사례 / 질문 |
| 상태 | 초안 / 발행 / 분석완료 |
| 반응 | 좋아요/RT/댓글 수 |
| 메모 | 배운 점 |
