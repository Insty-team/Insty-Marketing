# 한국어 DM 템플릿 (세그먼트별)

> 타겟: 한국 솔로프리너 / 1인 사업자
> 채널: X DM, 링크드인 메시지, 오픈카톡

---

## 세그먼트 1: 프리랜서 (개발자, 디자이너, 마케터)

### 첫 DM
```
안녕하세요 {이름}님! {최근 프로젝트/포트폴리오}를 보고 연락드려요.

프리랜서로 활동하시면서 고객 유치는 어떻게 하고 계세요?
저도 프리랜서 분들과 많이 일하는데, 실력은 좋은데 "다음 고객"을
찾는 데 에너지를 너무 많이 쓰시는 분이 많더라고요.

혹시 10분 정도 세일즈 파이프라인 무료 진단 받아보실 의향 있으세요?
지금 어디서 고객이 빠지고 있는지 바로 보여드릴 수 있어요.
```

### 팔로업 (48시간 후)
```
{이름}님, 혹시 메시지 보셨을까요?
프리랜서 분들 세일즈 자동화 사례 하나 공유드릴게요:

→ 디자이너 분이 월 3시간만 투자해서 인바운드 문의가 2배 늘어난 케이스예요.
관심 있으시면 말씀해주세요!
```

## 세그먼트 2: 코치/컨설턴트

### 첫 DM
```
안녕하세요 {이름}님! {강의/콘텐츠/프로필 언급} 인상 깊게 봤습니다.

코칭 하시면서 새 클라이언트 확보는 주로 어떻게 하고 계세요?
요즘 AI로 리드 발굴부터 미팅 예약까지 자동화하는 분들이 늘고 있는데,

혹시 관심 있으시면 {이름}님 비즈니스에 맞는 무료 파이프라인 진단을
15분 정도 해드릴 수 있어요. 부담 없이 말씀해주세요!
```

### 팔로업 (48시간 후)
```
{이름}님, 추가로 하나만 여쭤볼게요 —
지금 신규 고객 유치에 주당 몇 시간 쓰세요?

그 시간을 절반으로 줄이면서 오히려 문의가 늘어나는 시스템이 있는데,
1주일이면 만들어드릴 수 있어요. 관심 있으시면 답해주세요!
```

## 세그먼트 3: 콘텐츠 크리에이터

### 첫 DM
```
안녕하세요 {이름}님! {채널명/콘텐츠} 잘 보고 있습니다.

콘텐츠 정말 좋은데, 혹시 이걸 매출로 연결하는 시스템은 있으세요?
팔로워/구독자가 있는데 수익화가 아쉬운 크리에이터 분들이 많더라고요.

AI로 팔로워 → 리드 → 고객 전환 파이프라인을 만들어드리는 서비스를 하고 있는데,
무료 진단 한 번 받아보시겠어요? 15분이면 됩니다.
```

### 팔로업 (48시간 후)
```
{이름}님, 비슷한 채널 규모({팔로워 수}) 크리에이터 분이
파이프라인 만든 후 첫 달에 유료 전환 {X}건 나온 사례가 있어요.

혹시 궁금하시면 사례 공유드릴게요!
```

## 세그먼트 4: 온라인 강사 (클래스101, 탈잉, 인프런)

### 첫 DM
```
안녕하세요 {이름}님! {강의명} 강의 보고 연락드려요.

혹시 강의 외에 1:1 코칭이나 컨설팅도 하고 계세요?
강의 수강생을 고가 서비스 고객으로 전환하는 파이프라인을
AI로 만들어드리는 서비스를 하고 있어요.

관심 있으시면 무료 진단 한 번 해드릴까요?
강의 수강생 → 문의 → 상담 → 계약으로 이어지는 흐름을 설계해드려요.
```

## 세그먼트 5: 스몰비즈 (쇼핑몰, 1인 에이전시)

### 첫 DM
```
안녕하세요 {이름}님! {비즈니스/제품} 잘 봤습니다.

사업하시면서 신규 고객 확보가 가장 힘드시지 않나요?
AI로 잠재 고객 발굴 + 자동 연락 + 고객 관리를
하나의 시스템으로 만들어드리는 서비스를 하고 있어요.

무료로 15분 진단 해드리는데,
어디서 고객이 빠지고 있는지 바로 보여드릴 수 있어요.
관심 있으시면 편하게 답 주세요!
```

## 커뮤니티/오픈카톡 게시글 템플릿

### 가치 제공형 (자기소개 + 무료 오퍼)
```
안녕하세요! 1인 사업자분들 AI 세일즈 파이프라인 만들어드리는 일을 하고 있습니다.

직접 사업하면서 느낀 건데, 실력 좋은 분들이
"고객을 찾는 것" 자체에 너무 많은 시간을 쓰더라고요.

그래서 AI(Claude)로 1주일 안에 맞춤 세일즈 파이프라인을 구축해드리는
서비스를 시작했습니다.

- 리드 자동 발굴
- SNS 콘텐츠 자동화
- DM/이메일 아웃리치 자동화
- Notion CRM

지금 론칭 초기라 무료 파이프라인 진단을 해드리고 있어요.
15분이면 되고, 지금 어디서 고객이 빠지고 있는지 바로 보여드립니다.

관심 있으신 분은 DM 주세요!
```

### 사례 공유형 (결과 후)
```
[파이프라인 구축 결과 공유]

{업종} 하시는 분 세일즈 파이프라인을 만들어드렸는데,
Before → After가 이렇습니다:

- 고객 유치 시간: 주 10시간 → 3시간
- 월 신규 문의: 3건 → 8건
- 사용 도구: 6개 → 2개 (Notion + AI)

궁금하신 분 있으시면 자세한 사례 공유해드릴게요.
무료 진단도 가능합니다!
```
