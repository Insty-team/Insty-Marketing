# 클라이언트 프로젝트 템플릿

> 새 클라이언트 시작 시 이 폴더를 `clients/{이름}/`으로 복사해서 사용

## 폴더 구조
```
{클라이언트명}/
├── README.md           ← 프로젝트 개요 + 진행 상태
├── research.md         ← Research 팀 리서치 결과
├── pipeline-design.md  ← 파이프라인 설계서
├── lead-system.md      ← 리드 발굴 시스템 문서
├── content-strategy.md ← 콘텐츠 전략 + 샘플
├── outreach/           ← 아웃리치 시퀀스
│   ├── dm-sequence.md
│   └── email-sequence.md
├── crm-setup.md        ← Notion CRM 설계
├── handover.md         ← 인수인계 가이드
└── feedback.md         ← 클라이언트 피드백 + 사례 스터디 소스
```
