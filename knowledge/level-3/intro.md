# 레벨 3 — 중급

## 학습 목표

- Hooks로 Claude Code의 동작을 자동화한다
- MCP 서버로 외부 도구와 연결한다
- 멀티 모델 전략으로 비용을 최적화한다
- 테스트 자동화 및 CI/CD 파이프라인에 통합한다
- 대규모 코드베이스를 효율적으로 다룬다

## 챕터 목록

1. Hooks 시스템
2. MCP 서버 연결
3. 멀티 모델 전략
4. 비용 최적화
5. 테스트 자동화
6. CI/CD 통합
7. 대규모 코드베이스
8. 트러블슈팅
9. 프롬프트 전략
10. Remote Control
