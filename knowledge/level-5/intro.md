# 레벨 5 -- 마스터

이 레벨을 마치면 멀티에이전트 시스템, Agent SDK 심화, Extended Thinking, 프로덕션 배포까지 기업급 AI 인프라를 구축할 수 있습니다.

## 학습 목표

- 멀티에이전트 Co-work 아키텍처를 설계하고 구현한다
- Agent SDK 심화 패턴으로 복잡한 워크플로우를 자동화한다
- Extended Thinking으로 복잡한 추론 문제를 해결한다
- Claude API 패턴으로 프로덕션 수준의 시스템을 구축한다
- 기업 환경에 안전하게 배포하고 운영한다

## 챕터 목록

1. 멀티에이전트 Co-work
2. Agent SDK 심화
3. 커스텀 서브에이전트
4. Claude API 패턴
5. Extended Thinking
6. 프로덕션 배포
