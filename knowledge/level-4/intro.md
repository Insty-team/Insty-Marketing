# 레벨 4 -- 고급

이 레벨을 마치면 커스텀 Skills 개발, Headless 자동화, Agent SDK 기반 서브에이전트 패턴, 팀 협업 워크플로우를 설계할 수 있습니다.

## 학습 목표

- 커스텀 Skills를 만들어 Claude Code를 확장한다
- Headless 모드로 완전 자동화 파이프라인을 구현한다
- Agent SDK로 서브에이전트 오케스트레이션을 구축한다
- Docker 환경에서 안전하게 에이전트를 실행한다
- 팀 협업 패턴과 코드 리뷰 워크플로우를 확립한다

## 챕터 목록

1. 커스텀 Skills 만들기
2. Headless 자동화 모드
3. 보안 주의사항
4. Agent SDK 기초
5. 서브에이전트 패턴
6. Docker 환경
7. 팀 협업 패턴
8. 코드 리뷰 워크플로우
9. Worktree 병렬 개발
