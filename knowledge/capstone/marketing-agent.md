# 마케팅팀 에이전트

마케팅팀 에이전트는 **콘텐츠 생성, 캠페인 전략, SEO 최적화**를 담당합니다. 데이터 기반 의사결정과 브랜드 일관성을 유지하면서 고품질 마케팅 콘텐츠를 생산합니다.

## 마케팅 에이전트 구현

에이전트는 "B2B SaaS 마케팅 전략가"로 설정되며, 데이터 기반 의사결정을 우선시하고, 브랜드 일관성을 유지하며, 전환 중심 SEO를 수행합니다. 최대 12턴, Read/Write/Glob 도구를 사용합니다.

## 주요 마케팅 워크플로우

### 1. 콘텐츠 캘린더 생성

블로그, LinkedIn, 뉴스레터 등 멀티 채널 월간 콘텐츠 캘린더를 생성합니다.

### 2. SEO 콘텐츠 최적화

- 키워드 밀도 1.5~2.5% 유지
- H2/H3에 키워드 포함
- 메타 설명 160자 이하
- 내부 링크 3개 이상

### 3. 경쟁사 분석

포지셔닝, 콘텐츠 전략, 타겟 세그먼트, 차별화 기회를 구조적으로 분석합니다.

### 4. 캠페인 성과 분석

KPI 달성률, 채널별 ROI, 개선 영역, A/B 테스트 아이디어를 평가합니다.

## 콘텐츠 유형별 프롬프트 최적화

| 콘텐츠 유형 | 프롬프트 전략 |
|-----------|-------------|
| 블로그 | AIDA 구조 사용, 첫 문단에서 문제 제기 |
| SNS 캡션 | 플랫폼별 길이 제한 준수 |
| 이메일 | 제목 35자 이내, 개인화 포함 |
| 보도자료 | 역피라미드 구조, 인용 최소 1개 |
| 광고 카피 | 한 가지 혜택에 집중, CTA 명확 |

## 활용 팁

- 브랜드 가이드라인을 별도 문서로 저장해 시스템 프롬프트에 포함
- 성과 좋은 콘텐츠 샘플을 few-shot 예시로 제공
- 콘텐츠 생성 시 항상 SEO 점수와 가독성 점수를 함께 요청
