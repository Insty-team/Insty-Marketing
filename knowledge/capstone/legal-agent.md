# 법무팀 에이전트

법무팀 에이전트는 **계약 검토, 컴플라이언스 확인, 법적 리스크 분석**을 담당합니다. 법무 전문가가 아닌 팀도 계약의 위험 요소를 사전에 파악할 수 있도록 돕습니다.

> **중요**: 법무 에이전트의 분석은 **참고용**입니다. 법적 구속력이 있는 결정은 반드시 자격을 갖춘 변호사의 검토가 필요합니다.

에이전트는 읽기 전용(Read)으로만 동작하며, 문서 수정 도구는 차단됩니다.

## 주요 법무 워크플로우

### 1. 계약 리스크 검토

SaaS 계약서 등의 위험 조항을 고위험/중위험/저위험으로 분류하고 구체적 수정 권고를 제시합니다.

### 2. 개인정보 정책 컴플라이언스

GDPR, 현지 데이터 보호법 기준으로 데이터 수집, 보존 기간, 사용자 권리 등의 준수 여부를 확인합니다.

### 3. NDA 검토

기밀유지 범위, 기간, 집행 가능성을 분석합니다.

### 4. 규제 모니터링

새로운 규정이 기존 정책에 미치는 영향을 평가합니다.

## 안전 장치

고위험 키워드(소송, 손해배상, 독점금지 등) 감지 시 자동 에스컬레이션을 트리거합니다. 모든 분석에 변호사 상담 필요 면책 조항을 포함합니다.
